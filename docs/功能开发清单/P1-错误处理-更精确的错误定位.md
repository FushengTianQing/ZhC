# P1 错误处理 - 更精确的错误定位

## 功能概述

**功能名称**：错误信息包含精确的行列号  
**优先级**：P1  
**功能模块**：错误处理  
**创建日期**：2026-04-09

---

## 1. 需求分析

### 1.1 当前状态

ZhC 编译器已有基础的错误定位机制：

- `SourceLocation` 类（`src/zhc/errors/base.py`）已支持行列号
- 各阶段错误类（LexerError、ParserError、SemanticError）已继承 `ZHCError`
- 错误消息格式化已包含位置信息

**现有问题**：
1. 词法分析阶段 Token 的行列号信息可能不完整
2. 语法分析阶段 AST 节点的行列号传递不完整
3. 语义分析阶段错误定位精度不足
4. 多行错误（如未闭合的括号）缺少结束位置信息
5. 缺少源码片段展示功能

### 1.2 目标状态

实现精确到字符级别的错误定位：

```
test.zhc:10:15: 错误[E001]: 未定义的变量 '计数器'
  |
10|     整数型 计数器 = 计数器 + 1;
  |               ^^^^^^^
  |               变量 '计数器' 在此处首次使用，但尚未声明
建议: 请检查变量是否已声明，或添加声明语句
```

### 2.3 各阶段增强方案

#### 2.3.1 词法分析阶段

**修改文件**：`src/zhc/parser/lexer.py`

```python
class Token:
    """Token - 增强位置信息"""
    
    def __init__(self, type_, value, line, column, end_column=None):
        self.type = type_
        self.value = value
        self.line = line
        self.column = column
        self.end_column = end_column or (column + len(str(value)))
        # 新增：记录 Token 文本
        self.text = str(value)
```

**实现要点**：
1. 每个 Token 记录精确的起始和结束列号
2. 多行 Token（如字符串）记录起始和结束行号
3. Token 生成时同步计算位置信息

#### 2.3.2 语法分析阶段

**修改文件**：`src/zhc/parser/ast_nodes.py`

```python
class ASTNode:
    """AST 节点 - 增强位置信息"""
    
    def __init__(self, node_type, line=0, column=0, end_line=None, end_column=None):
        self.node_type = node_type
        self.line = line
        self.column = column
        self.end_line = end_line
        self.end_column = end_column
        # 新增：记录节点文本范围
        self.token_range = None  # (start_token_idx, end_token_idx)
```

**实现要点**：
1. Parser 从 Token 流中提取位置信息
2. 复合节点（如函数声明）计算整体范围
3. 错误节点记录期望位置和实际位置

#### 2.3.3 语义分析阶段

**修改文件**：`src/zhc/semantic/semantic_analyzer.py`

```python
class SemanticAnalyzer:
    """语义分析器 - 增强位置追踪"""
    
    def report_error(
        self,
        error_type: str,
        message: str,
        node: ASTNode,
        suggestion: str = None
    ):
        """报告语义错误"""
        location = SourceLocation(
            file_path=self.current_file,
            line=node.line,
            column=node.column,
            end_line=node.end_line,
            end_column=node.end_column,
            token_text=self._get_node_text(node)
        )
        error = SemanticError(
            message=message,
            location=location,
            error_code=error_type,
            suggestion=suggestion
        )
        self.errors.add(error)
```
