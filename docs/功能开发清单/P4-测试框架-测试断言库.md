# P4-测试框架-测试断言库

## 功能概述

**优先级**: P4  
**功能模块**: 测试框架  
**功能名称**: 测试断言库  
**功能描述**: 提供丰富的断言函数用于测试验证  
**预估工时**: 3天  
**依赖项**: 单元测试支持  
**状态**: 待开发

---

## 1. 需求分析

实现完整的断言库，支持各种类型的断言：

| 分类 | 断言函数 | 说明 |
|-----|---------|------|
| 基本 | 断言等于(a, b) | 断言两个值相等 |
| 比较 | 断言大于(a, b) | a > b |
| 布尔 | 断言为真(条件) | 断言条件为真 |
| 指针 | 断言为空(ptr) | ptr == NULL |
| 字符串 | 断言字符串等于(a, b) | strcmp(a, b) == 0 |
| 浮点 | 断言浮点等于(a, b, eps) | fabs(a-b) < eps |
| 数组 | 断言数组相等(a, b, n) | memcmp(a, b, n) == 0 |

## 2. 实现方案

### 2.1 断言库头文件

```c
// src/lib/zhc_test.h

#ifndef ZHC_TEST_H
#define ZHC_TEST_H

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <math.h>

// 全局统计
typedef struct {
    int total;
    int passed;
    int failed;
    int skipped;
} TestStats;

extern TestStats g_stats;

// 基本断言
#define 断言等于(a, b) \
    _zhc_assert_eq(__FILE__, __LINE__, #a, #b, (a), (b))

#define 断言不等于(a, b) \
    _zhc_assert_ne(__FILE__, __LINE__, #a, #b, (a), (b))

#define 断言大于(a, b) \
    _zhc_assert_gt(__FILE__, __LINE__, #a, #b, (a), (b))

#define 断言小于(a, b) \
    _zhc_assert_lt(__FILE__, __LINE__, #a, #b, (a), (b))

// 布尔断言
#define 断言为真(条件) \
    _zhc_assert_true(__FILE__, __LINE__, #条件, (条件))

#define 断言为假(条件) \
    _zhc_assert_false(__FILE__, __LINE__, #条件, (条件))

// 指针断言
#define 断言为空(ptr) \
    _zhc_assert_null(__FILE__, __LINE__, #ptr, (ptr))

#define 断言非空(ptr) \
    _zhc_assert_not_null(__FILE__, __LINE__, #ptr, (ptr))

// 字符串断言
#define 断言字符串等于(a, b) \
    _zhc_assert_str_eq(__FILE__, __LINE__, #a, #b, (a), (b))

#define 断言字符串包含(s, sub) \
    _zhc_assert_str_contains(__FILE__, __LINE__, #s, #sub, (s), (sub))

// 浮点断言
#define 断言浮点等于(a, b, epsilon) \
    _zhc_assert_float_eq(__FILE__, __LINE__, #a, #b, (a), (b), (epsilon))

// 内存断言
#define 断言数组相等(a, b, n) \
    _zhc_assert_mem_eq(__FILE__, __LINE__, #a, #b, #n, (a), (b), (n))

// 辅助函数
int _zhc_assert_eq(const char* file, int line, const char* a_str, const char* b_str, int a, int b);
int _zhc_assert_true(const char* file, int line, const char* cond_str, int cond);
int _zhc_assert_null(const char* file, int line, const char* ptr_str, const void* ptr);
int _zhc_assert_str_eq(const char* file, int line, const char* a_str, const char* b_str, const char* a, const char* b);
int _zhc_assert_float_eq(const char* file, int line, const char* a_str, const char* b_str, double a, double b, double epsilon);
int _zhc_assert_mem_eq(const char* file, int line, const char* a_str, const char* b_str, const char* n_str, const void* a, const void* b, size_t n);

#endif
```
