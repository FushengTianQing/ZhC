## 1. 需求分析

### 1.1 功能目标

实现文档注释解析器：

1. **多格式支持**: 支持 Doxygen、Javadoc 等格式
2. **标签解析**: 解析 @param, @return 等标签
3. **Markdown 支持**: 支持注释中的 Markdown 语法
4. **代码示例提取**: 提取注释中的代码示例

### 1.2 支持的标签

| 标签 | 说明 | 示例 |
|-----|------|------|
| @param | 参数说明 | @param a 第一个参数 |
| @return | 返回值说明 | @return 计算结果 |
| @示例 | 代码示例 | @示例 加(1, 2) |
| @注意 | 注意事项 | @注意 可能溢出 |
| @参见 | 参见其他 | @参见 减函数 |
| @版本 | 版本信息 | @版本 1.0 |

## 2. 实现方案

### 2.1 注释解析器

```python
# src/doc/comment_parser.py

from typing import Dict, List, Optional
from dataclasses import dataclass
import re

@dataclass
class DocTag:
    """文档标签"""
    name: str
    value: str
    params: Dict[str, str]

@dataclass
class DocComment:
    """文档注释"""
    description: str
    tags: List[DocTag]
    examples: List[str]
    notes: List[str]
    
    def get_param(self, name: str) -> Optional[str]:
        """获取参数说明"""
        for tag in self.tags:
            if tag.name == 'param' and tag.params.get('name') == name:
                return tag.value
        return None
    
    def get_return(self) -> Optional[str]:
        """获取返回值说明"""
        for tag in self.tags:
            if tag.name == 'return':
                return tag.value
        return None

class CommentParser:
    """注释解析器"""
    
    TAG_PATTERN = re.compile(r'@(\w+)\s*(.*)')
    
    def parse(self, comment_text: str) -> DocComment:
        """解析文档注释"""
        lines = comment_text.strip().split('\n')
        
        description_lines = []
        tags = []
        examples = []
        notes = []
        
        current_tag = None
        
        for line in lines:
            # 移除注释标记
            line = line.strip().lstrip('*').strip()
            
            # 检查标签
            match = self.TAG_PATTERN.match(line)
            if match:
                tag_name = match.group(1)
                tag_value = match.group(2).strip()
                
                # 解析标签参数
                params = {}
                if tag_name == 'param':
                    parts = tag_value.split(None, 1)
                    if len(parts) >= 1:
                        params['name'] = parts[0]
                        tag_value = parts[1] if len(parts) > 1 else ''
                
                tag = DocTag(name=tag_name, value=tag_value, params=params)
                tags.append(tag)
                
                if tag_name == '示例':
                    examples.append(tag_value)
                elif tag_name == '注意':
                    notes.append(tag_value)
            else:
                # 描述文本
                if not tags:
                    description_lines.append(line)
        
        return DocComment(
            description='\n'.join(description_lines),
            tags=tags,
            examples=examples,
            notes=notes
        )
```
