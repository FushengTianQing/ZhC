## 1. 需求分析

### 1.1 功能目标

实现代码覆盖率分析工具：

1. **行覆盖率**: 统计执行的代码行数
2. **分支覆盖率**: 统计条件分支的执行情况
3. **函数覆盖率**: 统计函数调用情况
4. **覆盖率报告**: 生成详细的覆盖率报告

### 1.2 覆盖率指标

| 指标 | 说明 | 公式 |
|-----|------|------|
| 行覆盖率 | 执行的代码行数 | 已执行行数 / 总行数 |
| 分支覆盖率 | 条件分支执行情况 | 已执行分支数 / 总分支数 |
| 函数覆盖率 | 函数调用情况 | 已调用函数数 / 总函数数 |

---

## 2. 实现方案

### 2.1 覆盖率工具架构

```
源代码 → 编译器(插桩) → 可执行文件 → 执行 → 覆盖率数据 → 报告生成
```

### 2.2 插桩实现

```c
// src/lib/zhc_coverage.h

#ifndef ZHC_COVERAGE_H
#define ZHC_COVERAGE_H

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <stdint.h>

// 覆盖率数据结构
typedef struct {
    uint64_t* line_hits;      // 行执行计数
    uint64_t* branch_hits;    // 分支执行计数
    uint64_t* function_hits; // 函数调用计数
    int line_count;
    int branch_count;
    int function_count;
} CoverageData;

extern CoverageData g_coverage;

// 初始化覆盖率跟踪
void coverage_init(const char* source_file);
void coverage_shutdown(void);

// 行覆盖率
#define COVERAGE_LINE(line_num) \
    do { \
        if (g_coverage.line_hits && (line_num) < g_coverage.line_count) { \
            g_coverage.line_hits[line_num]++; \
        } \
    } while(0)

// 分支覆盖率
#define COVERAGE_BRANCH(branch_id, taken) \
    do { \
        if (g_coverage.branch_hits && (branch_id) < g_coverage.branch_count) { \
            g_coverage.branch_hits[branch_id * 2 + (taken)]++; \
        } \
    } while(0)

// 函数覆盖率
#define COVERAGE_FUNCTION(func_id) \
    do { \
        if (g_coverage.function_hits && (func_id) < g_coverage.function_count) { \
            g_coverage.function_hits[func_id]++; \
        } \
    } while(0)

// 报告生成
void coverage_report(const char* output_file);
void coverage_print_summary(void);

#endif
```
