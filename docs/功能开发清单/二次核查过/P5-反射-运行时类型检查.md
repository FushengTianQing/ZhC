## 1. 功能概述

实现在运行时检查对象类型、判断类型兼容性、执行类型断言，支持多态和动态类型验证。

### 1.1 目标

- ✅ 支持类型兼容性检查
- ✅ 支持类型断言（cast）
- ✅ 支持接口实现检查
- ✅ 支持继承关系检查

### 1.2 语法设计

```zhc
// 类型检查
类 动物 {
    函数型 无 -> 无 叫声;
}

类 狗 继承 动物 {
    函数 叫() {
        打印("汪汪!");
    }
}

// 类型兼容性检查
动物 a = 新 狗();
如果 (是类型(狗, a)) {
    打印("a 是狗");
}

// 安全类型转换
狗 d = 安全转换[狗](a);

// 接口检查
如果 (实现接口(可飞行, 对象)) {
    打印("对象可以飞");
}

// 类型名称获取
字符串 类型名 = 类型名称(对象);
```

## 5. 设计亮点

1. **复用 TypeRegistry**：所有类型查询基于已有的 TypeRegistry，无需维护额外状态
2. **继承链 + 接口双路径**：`is_a()` 同时检查继承链和接口实现
3. **多层转换策略**：safe_cast（返回 None）/ dynamic_cast（抛异常）/ require_type（断言）
4. **完整 MRO**：方法解析顺序支持多级继承
5. **C 运行时链表结构**：与 zhc_reflection.h 一致的设计风格
