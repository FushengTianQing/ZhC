> **优先级**: P5
> **功能模块**: 内存管理

> **创建日期**: 2026-04-10
> **状态**: 规划中

## 1. 功能概述

实现运行时内存泄漏检测功能，追踪内存分配和释放，检测泄漏的内存块，生成泄漏报告并定位泄漏位置。

### 1.1 目标

- 追踪所有内存分配和释放
- 检测未释放的内存块
- 生成详细的泄漏报告
- 定位泄漏源代码位置

### 1.2 语法设计

```zhc
// 启用内存泄漏检测
启用内存检测();

// 内存操作
整数型* ptr = 分配[整数型](100);
释放(ptr);

// 手动检测
内存检测.检测泄漏();

// 生成报告
字符串 报告 = 内存检测.生成报告();
打印(报告);

// 泄漏检测示例
函数 可能泄漏() {
    整数型* data = 分配[整数型](1000);
    // 忘记释放 data
    返回;
}

函数 正确使用() {
    整数型* data = 分配[整数型](1000);
    使用数据(data);
    释放(data);  // 正确释放
    返回;
}
```

### 1.3 报告格式

```
=== 内存泄漏报告 ===
时间: 2026-04-10 12:00:00

总分配: 5 次
总释放: 3 次
泄漏: 2 次
泄漏大小: 2048 字节

=== 泄漏详情 ===

泄漏 #1:
  地址: 0x7f9a3c0008c0
  大小: 1024 字节
  位置: test.zhc:10
  分配调用栈:
    #0: test.zhc:10 (分配)
    #1: test.zhc:15 (可能泄漏函数)
    #2: main.zhc:30

泄漏 #2:
  地址: 0x7f9a3c0009d0
  大小: 1024 字节
  位置: test.zhc:25
  类型: 整数型数组
```

### 3.2 核心数据结构

```python
# src/zhc/memory/leak_detector.py

from dataclasses import dataclass, field
from typing import Dict, List, Optional, Set, Tuple
from enum import Enum
from collections import defaultdict

@dataclass
class AllocationInfo:
    """分配信息"""
    address: int
    size: int
    type_name: Optional[str] = None
    line_number: Optional[int] = None
    file_name: Optional[str] = None
    function_name: Optional[str] = None
    call_stack: List[Tuple[str, int, str]] = field(default_factory=list)

@dataclass
class LeakReport:
    """泄漏报告"""
    address: int
    size: int
    type_name: Optional[str]
    location: str
    call_stack: List[Tuple[str, int, str]]
    severity: str
    description: str
    suggested_fix: str

class LeakDetector:
    """内存泄漏检测器"""
    
    def __init__(self):
        self.allocations: Dict[int, AllocationInfo] = {}
        self.freed: Set[int] = set()
        self.leaks: List[LeakReport] = []
        self.total_allocated: int = 0
        self.total_freed: int = 0
        
    def track_allocation(self, address: int, size: int, **kwargs) -> None:
        """追踪内存分配"""
        info = AllocationInfo(address=address, size=size, **kwargs)
        self.allocations[address] = info
        self.total_allocated += 1
    
    def track_free(self, address: int) -> None:
        """追踪内存释放"""
        if address in self.allocations:
            del self.allocations[address]
            self.total_freed += 1
    
    def detect_leaks(self) -> List[LeakReport]:
        """检测泄漏"""
        self.leaks = []
        for address, info in self.allocations.items():
            report = LeakReport(
                address=address,
                size=info.size,
                type_name=info.type_name,
                location=f"{info.file_name}:{info.line_number}",
                call_stack=info.call_stack,
                severity=self._calculate_severity(info.size),
                description=f"未释放内存: {info.size} 字节",
                suggested_fix="在适当位置添加释放代码",
            )
            self.leaks.append(report)
        return self.leaks
    
    def _calculate_severity(self, size: int) -> str:
        if size > 1024 * 1024: return "critical"
        elif size > 100 * 1024: return "high"
        elif size > 10 * 1024: return "medium"
        return "low"
```
