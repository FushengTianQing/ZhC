## 1. 需求分析

### 1.1 功能目标

提供 ZhC 标准库函数，支持不同字符编码之间的转换：

1. **UTF-8 ↔ 其他编码**: GBK、GB2312、BIG5、Shift-JIS 等
2. **Unicode 规范化**: NFC、NFD、NFKC、NFKD
3. **编码检测**: 自动检测字符串的编码

**示例**:
```zhc
外部 函数 字节数组型 转码(字节数组型 输入, 字符串型 源编码, 字符串型 目标编码);
外部 函数 字符串型 到UTF8(字节数组型 数据, 字符串型 源编码);
外部 函数 字节数组型 从UTF8(字符串型 输入, 字符串型 目标编码);
```

### 1.2 常用编码

| 编码名称 | 说明 | 使用地区 |
|---------|------|---------|
| UTF-8 | Unicode 转换格式 | 全球 |
| GBK | 汉字扩展 | 中国大陆 |
| GB2312 | 简体中文字符集 | 中国大陆 |
| BIG5 | 繁体中文 | 台湾、香港 |
| Shift-JIS | 日文字符集 | 日本 |
| EUC-KR | 韩文字符集 | 韩国 |
| ISO-8859-1 | 拉丁字符 | 西欧 |
| Windows-1252 | 扩展 Latin | Windows |

#### 步骤2：实现底层转换

使用 Python 的 `codecs` 模块或 C 库如 `iconv`:

```c
// encoding.c - 底层编码转换实现
#include <iconv.h>
#include <stdlib.h>
#include <string.h>

// 转换函数
int convert_encoding(
    const char* input, size_t input_len,
    const char* from_encoding,
    const char* to_encoding,
    char** output, size_t* output_len
) {
    iconv_t cd = iconv_open(to_encoding, from_encoding);
    if (cd == (iconv_t)-1) {
        return -1;
    }
    
    size_t in_left = input_len;
    size_t out_left = input_len * 4;  // 预估最大长度
    char* outbuf = malloc(out_left);
    char* outptr = outbuf;
    
    size_t result = iconv(cd, &input, &in_left, &outptr, &out_left);
    iconv_close(cd);
    
    if (result == (size_t)-1) {
        free(outbuf);
        return -1;
    }
    
    *output = outbuf;
    *output_len = outptr - outbuf;
    return 0;
}
```

}
