## 1. 开发内容分析

### 1.1 目标概述

实现完整的调试功能，包括断点管理、变量查看和调用栈分析，为开发者提供完整的源码级调试体验。

### 1.2 技术背景

#### 调试信息流程
```
源文件行号 → DWARF行号表 → 地址映射 → 断点设置
                          ↓
                     运行时断点命中
                          ↓
                     变量位置查找
                          ↓
                     变量值读取 → 美化显示
```

### 1.3 需求分析

#### 核心需求
1. **断点管理**：设置、删除、启用、禁用断点
2. **变量查看**：查看局部变量、全局变量、寄存器
3. **调用栈**：显示调用栈、切换栈帧
4. **表达式求值**：计算表达式值

#### 断点引擎
```python
class BreakpointEngine:
    """断点引擎"""

    def set_source_breakpoint(self, file: str, line: int) -> int:
        """设置源码断点"""
        # 1. 使用行号查找地址
        addr = self._lookup_address(file, line)

        # 2. 设置软件断点
        breakpoint_id = self._set_breakpoint(addr)

        # 3. 注册断点回调
        self._register_breakpoint_callback(breakpoint_id, file, line)

        return breakpoint_id

    def set_watchpoint(self, var_name: str, watch_type: WatchType) -> int:
        """设置监视点"""
        # 1. 获取变量地址
        var_addr = self._get_variable_address(var_name)

        # 2. 设置监视点
        if watch_type == WatchType.WRITE:
            return self._set_write_watchpoint(var_addr)
        elif watch_type == WatchType.READ:
            return self._set_read_watchpoint(var_addr)
        else:
            return self._set_access_watchpoint(var_addr)
```

#### 变量打印器
```python
class VariablePrinter:
    """变量打印器"""

    def print_local_variables(self, frame: Frame) -> Dict[str, VariableValue]:
        """打印局部变量"""
        variables = {}

        # 遍历帧的词法作用域
        for scope in frame.scopes:
            for var in scope.variables:
                if self._is_in_scope(var, frame.pc):
                    variables[var.name] = self._read_variable(var, frame)

        return variables

    def _read_variable(self, var: Variable, frame: Frame) -> VariableValue:
        """读取变量值"""
        location = var.location

        if isinstance(location, RegisterLocation):
            return self._read_register(location.reg, frame)

        elif isinstance(location, StackLocation):
            return self._read_stack(location.offset, var.type, frame)

        elif isinstance(location, MemoryLocation):
            return self._read_memory(location.address, var.type, frame)

        elif isinstance(location, ConstantLocation):
            return VariableValue(value=location.value, type=var.type)

        raise DebuggerError(f"Unknown location type: {location}")
```

#### 栈帧分析器
```python
class StackFrameAnalyzer:
    """栈帧分析器"""

    def get_backtrace(self, thread: Thread) -> List[Frame]:
        """获取调用栈"""
        frames = []

        # 从当前帧开始向上遍历
        current_frame = thread.frame
        while current_frame:
            # 解析帧信息
            frame_info = self._analyze_frame(current_frame)

            # 获取函数名和位置
            func_info = self._get_function_info(current_frame.pc)

            frames.append(Frame(
                number=len(frames),
                pc=current_frame.pc,
                function=func_info.name,
                file=func_info.file,
                line=func_info.line,
                arguments=self._get_arguments(current_frame),
                locals=self._get_locals(current_frame),
            ))

            # 移动到调用者帧
            current_frame = current_frame.caller

        return frames

    def _get_function_info(self, pc: int) -> FunctionInfo:
        """获取函数信息"""
        # 查询 DWARF
        unit = self._find_compile_unit(pc)
        die = self._find_function_die(unit, pc)

        return FunctionInfo(
            name=die.get_name(),
            file=unit.get_file(die.get_file_index()),
            line=unit.get_line_for_address(pc),
        )
```

#### 表达式解析器
```python
class ExpressionEvaluator:
    """表达式求值器"""

    def evaluate(self, expr: str, frame: Frame) -> VariableValue:
        """求值表达式"""
        # 1. 解析表达式
        ast = self._parse_expression(expr)

        # 2. 语义分析（变量绑定）
        bound = self._bind_variables(ast, frame)

        # 3. 求值
        return self._evaluate(bound, frame)

    def _evaluate_member_access(self, expr: MemberAccess,
                                frame: Frame) -> VariableValue:
        """求值成员访问"""
        # 获取基对象
        base = self._evaluate(expr.base, frame)

        # 查找成员
        member = base.type.get_member(expr.member_name)

        # 读取成员值
        offset = member.offset
        return self._read_memory(base.address + offset, member.type, frame)
```
