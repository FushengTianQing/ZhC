## 1. 需求分析

### 1.1 功能目标

实现热点分析工具：

1. **热点识别**: 自动识别耗时最多的函数
2. **调用路径分析**: 分析热点函数的调用路径
3. **优化建议**: 提供优化建议
4. **可视化**: 生成火焰图等可视化报告

### 1.2 分析指标

| 指标 | 说明 |
|-----|------|
| 执行时间占比 | 函数执行时间占总时间的百分比 |
| 调用次数 | 函数被调用的次数 |
| 平均执行时间 | 单次调用的平均执行时间 |
| 调用深度 | 函数在调用栈中的深度 |

## 2. 实现方案

### 2.1 热点分析器

```python
# src/analyzer/hotspot_analyzer.py

from typing import List, Dict
from dataclasses import dataclass

@dataclass
class Hotspot:
    """热点函数"""
    function_name: str
    total_time: float
    percentage: float
    call_count: int
    avg_time: float
    optimization_hints: List[str]

class HotspotAnalyzer:
    """热点分析器"""
    
    def __init__(self, profile_data: Dict):
        self.profile_data = profile_data
        self.hotspots = []
    
    def analyze(self, threshold: float = 5.0) -> List[Hotspot]:
        """分析热点函数
        
        Args:
            threshold: 时间占比阈值（百分比）
        
        Returns:
            热点函数列表
        """
        total_time = sum(f['total_time'] for f in self.profile_data.values())
        
        for func_name, data in self.profile_data.items():
            percentage = (data['total_time'] / total_time) * 100
            
            if percentage >= threshold:
                hotspot = Hotspot(
                    function_name=func_name,
                    total_time=data['total_time'],
                    percentage=percentage,
                    call_count=data['call_count'],
                    avg_time=data['total_time'] / data['call_count'],
                    optimization_hints=self._generate_hints(data)
                )
                self.hotspots.append(hotspot)
        
        # 按时间占比排序
        self.hotspots.sort(key=lambda x: x.percentage, reverse=True)
        
        return self.hotspots
    
    def _generate_hints(self, data: Dict) -> List[str]:
        """生成优化建议"""
        hints = []
        
        # 调用次数过多
        if data['call_count'] > 10000:
            hints.append("考虑减少调用次数或使用缓存")
        
        # 单次执行时间过长
        avg_time = data['total_time'] / data['call_count']
        if avg_time > 1.0:  # 超过 1 秒
            hints.append("单次执行时间过长，考虑优化算法")
        
        return hints
    
    def generate_report(self) -> str:
        """生成分析报告"""
        lines = ["=== 热点分析报告 ===\n"]
        
        for i, hotspot in enumerate(self.hotspots, 1):
            lines.append(f"{i}. {hotspot.function_name}")
            lines.append(f"   时间占比: {hotspot.percentage:.2f}%")
            lines.append(f"   总时间: {hotspot.total_time:.3f}s")
            lines.append(f"   调用次数: {hotspot.call_count}")
            lines.append(f"   平均时间: {hotspot.avg_time:.6f}s")
            
            if hotspot.optimization_hints:
                lines.append("   优化建议:")
                for hint in hotspot.optimization_hints:
                    lines.append(f"     - {hint}")
            
            lines.append("")
        
        return '\n'.join(lines)
```
