## 1. 需求分析

### 1.1 功能目标

实现内置单元测试框架：

1. **测试用例**: 支持测试函数的定义和执行
2. **测试套件**: 支持测试分组和组织
3. **测试运行器**: 支持测试发现和执行
4. **断言库**: 提供丰富的断言函数
5. **测试隔离**: 确保测试之间相互独立

**示例语法**:
```zhc
测试模块 "数学库测试"
导入 数学模块

测试套件 "算术运算测试"

测试函数 测试加法() {
    整数型 结果 = 数学模块.加(2, 3);
    断言等于(结果, 5, "2 + 3 应该等于 5");
}

主函数() {
    测试结果 结果 = 运行测试(测试加法);
    打印结果(结果);
    返回 结果.是否通过() ? 0 : 1;
}
```

### 1.2 测试框架规范

| 断言函数 | 说明 | 示例 |
|---------|------|------|
| 断言等于(a, b) | 断言相等 | 断言等于(x, 5) |
| 断言不等于(a, b) | 断言不等 | 断言不等于(x, 0) |
| 断言为真(条件) | 断言为真 | 断言为真(x > 0) |
| 断言为假(条件) | 断言为假 | 断言为假(x < 0) |
| 断言为空(指针) | 断言为空 | 断言为空(ptr) |
| 断言非空(指针) | 断言非空 | 断言非空(ptr) |
| 断言浮点等于(a, b, epsilon) | 浮点数比较 | 断言浮点等于(x, 3.14, 0.001) |

```c
// src/lib/zhc_test.h

#ifndef ZHC_TEST_H
#define ZHC_TEST_H

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <math.h>

// 全局计数器
extern int _assert_count;
extern int _pass_count;
extern int _fail_count;

// 报告断言结果
void _zhc_report_assertion(const char* file, int line, int passed, const char* msg);

// 断言宏
#define 断言等于(a, b, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, (a) == (b), #a " == " #b)

#define 断言不等于(a, b, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, (a) != (b), #a " != " #b)

#define 断言为真(条件, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, (条件), #条件)

#define 断言为假(条件, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, !(条件), "!(" #条件 ")")

#define 断言为空(指针, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, (指针) == NULL, #指针 " == NULL")

#define 断言非空(指针, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, (指针) != NULL, #指针 " != NULL")

#define 断言浮点等于(a, b, epsilon, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, fabs((a) - (b)) < (epsilon), #a " ≈ " #b)

#define 断言字符串等于(a, b, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, strcmp((a), (b)) == 0, #a " == " #b)

#define 断言大于(a, b, ...) \
    _zhc_report_assertion(__FILE__, __LINE__, (a) > (b), #a " > " #b)

    _zhc_report_assertion(__FILE__, __LINE__, (a) < (b), #a " < " #b)

#endif
```
