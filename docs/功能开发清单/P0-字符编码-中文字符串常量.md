## 1. 需求分析

### 1.1 功能目标

支持 ZhC 源代码中直接使用中文字符串常量，包括：
1. 基本中文字符串
2. 中文转义序列
3. 中文字符串与其他类型的混合使用

**示例**:
```zhc
整数型 主函数() {
    // 基本中文字符串
    字符串型 问候 = "你好";
    字符串型 消息 = "欢迎使用中文编程语言";
    
    // 带中文的格式化字符串
    打印("姓名：%s，年龄：%d\n", "张三", 25);
    
    // 中文错误消息
    字符串型 错误消息 = "错误：文件不存在";
    
    返回 0;
}
```

### 1.2 设计考量

**选项 A**: 仅支持 UTF-8 编码的中文字符串（与 C 语言兼容）
**选项 B**: 支持 ZhC 特有的中文字符串语法糖

**选择**: 选项 A（初期）+ 选项 B（后期）

**理由**:
1. 保持与 C 语言的兼容性
2. UTF-8 是现代编程语言的标准编码
3. 语法糖可以后续迭代
