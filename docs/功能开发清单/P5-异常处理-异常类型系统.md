> **优先级**: P5

> **创建日期**: 2026-04-10
> **状态**: 规划中

## 1. 功能概述

实现完整的异常类型系统，支持自定义异常类型、异常层次结构、异常构造和析构，为 try-catch 机制提供类型基础。

### 1.1 目标

- 定义内置异常类型层次
- 支持自定义异常类型继承
- 支持异常消息和错误码
- 支持异常链（cause）

### 1.2 语法设计

```zhc
// 定义异常类型
异常类 文件错误 继承 输入输出错误 {
    字符串 文件名;
    整数型 错误码;
    
    构造函数(字符串 文件名, 整数型 错误码) {
        本.文件名 = 文件名;
        本.错误码 = 错误码;
        本.消息 = "文件错误: " + 文件名;
    }
}

// 使用
抛出 新 文件错误("test.txt", 404);
```

### 1.3 内置异常类型层次

```
异常 (基类)
├── 错误 (Error - 不可恢复)
│   ├── 内存错误 (MemoryError)
│   ├── 栈溢出错误 (StackOverflowError)
│   └── 系统错误 (SystemError)
│
└── 异常 (Exception - 可恢复)
    ├── 运行时异常 (RuntimeException)
    │   ├── 空指针异常 (NullPointerException)
    │   ├── 数组越界异常 (ArrayIndexOutOfBoundsException)
    │   └── 类型转换异常 (ClassCastException)
    │
    ├── 输入输出异常 (IOException)
    │   ├── 文件未找到异常 (FileNotFoundException)
    │   └── 文件权限异常 (FilePermissionException)
    │
    ├── 算术异常 (ArithmeticException)
    │   ├── 除零异常 (DivideByZeroException)
    │   └── 溢出异常 (OverflowException)
    │
    └── 用户自定义异常
```

### 3.2 数据结构设计

```python
# src/zhc/exception/types.py

from dataclasses import dataclass, field
from typing import Dict, List, Optional, Any

@dataclass
class ExceptionField:
    """异常字段"""
    name: str
    type_name: str
    default_value: Optional[Any] = None

@dataclass
class ExceptionType:
    """异常类型定义"""
    name: str                    # 类型名称
    base_class: Optional[str]    # 父类名称
    fields: List[ExceptionField] # 字段列表
    methods: Dict[str, str]      # 方法名 -> 方法签名
    error_code: Optional[str]    # 关联的错误码
    is_builtin: bool = False     # 是否内置类型
    
    def is_subtype_of(self, other: str) -> bool:
        """检查是否为某类型的子类型"""
        ...

@dataclass
class ExceptionObject:
    """异常对象"""
    type_name: str               # 类型名称
    message: str                 # 错误消息
    error_code: Optional[int]    # 错误码
    cause: Optional['ExceptionObject'] = None  # 原因异常
    stack_trace: List[str] = field(default_factory=list)  # 堆栈跟踪
    fields: Dict[str, Any] = field(default_factory=dict)  # 字段值
```

### 3.3 类型注册表

```python
# src/zhc/exception/registry.py

class ExceptionRegistry:
    """异常类型注册表"""
    
    _instance: Optional['ExceptionRegistry'] = None
    _types: Dict[str, ExceptionType] = {}
    
    @classmethod
    def instance(cls) -> 'ExceptionRegistry':
        if cls._instance is None:
            cls._instance = cls()
            cls._instance._register_builtins()
        return cls._instance
    
    def register(self, exc_type: ExceptionType) -> None:
        """注册异常类型"""
        self._types[exc_type.name] = exc_type
    
    def lookup(self, name: str) -> Optional[ExceptionType]:
        """查找异常类型"""
        return self._types.get(name)
    
    def is_subtype(self, subtype: str, supertype: str) -> bool:
        """检查子类型关系"""
        if subtype == supertype:
            return True
        exc = self._types.get(subtype)
        if exc and exc.base_class:
            return self.is_subtype(exc.base_class, supertype)
        return False
    
    def _register_builtins(self) -> None:
        """注册内置异常类型"""
        # 注册 Exception 基类
        self.register(ExceptionType(
            name="异常",
            base_class=None,
            fields=[
                ExceptionField("消息", "字符串"),
                ExceptionField("错误码", "整数型"),
            ],
            methods={"打印": "函数 -> 无"},
            is_builtin=True,
        ))
        # ... 其他内置类型
```

## 5. API 设计

### 5.1 Python API

```python
# src/zhc/exception/__init__.py

from .types import ExceptionType, ExceptionObject, ExceptionField
from .registry import ExceptionRegistry
from .builtins import (
    BUILTIN_EXCEPTIONS,
    get_exception_class,
    is_exception_type,
)

# 注册自定义异常
registry = ExceptionRegistry.instance()
registry.register(ExceptionType(
    name="自定义错误",
    base_class="异常",
    fields=[
        ExceptionField("代码", "整数型"),
    ],
))

# 创建异常对象
exc = ExceptionObject(
    type_name="空指针异常",
    message="访问空指针",
    error_code=1001,
)
```

```c
// src/zhc/lib/zhc_exception.h
```
