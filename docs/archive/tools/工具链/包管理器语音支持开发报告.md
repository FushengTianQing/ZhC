## 功能概述

为中文C语言包管理器添加了完整的语音支持系统，包括**语音识别**、**命令解析**和**语音反馈**三大模块。

## 核心设计

### 1. 语音识别模块（SpeechRecognizer）

**位置**：`src/package/voice/speech_recognizer.py`

**核心功能**：
- 支持多种语音识别引擎（在线API/本地Whisper/本地Vosk/规则匹配）
- 多级降级策略（在线API → 本地模型 → 规则匹配）
- 统计信息追踪（请求次数、成功率、延迟等）
- 麦克风监听与文件识别

**支持的引擎类型**：
```python
class SpeechEngine(Enum):
    ONLINE_API = "在线API"      # 百度/讯飞/腾讯云等
    LOCAL_WHISPER = "本地Whisper"  # OpenAI Whisper
    LOCAL_VOSK = "本地Vosk"     # 离线语音识别
    PATTERN = "规则匹配"        # 兜底方案
```

**预定义语音命令模式**：
- 安装命令：安装、下载、获取
- 搜索命令：搜索、查找、找
- 列表命令：列出、显示、查看
- 卸载命令：卸载、删除、移除
- 更新命令：更新、升级
- 验证命令：验证、检查
- 帮助命令：帮助、怎么用

### 2. 命令解析模块（VoiceCommandParser）

**位置**：`src/package/voice/voice_command_parser.py`

**核心功能**：
- 自然语言命令解析
- 命令意图识别
- 参数提取（包名、版本等）
- 置信度计算
- 命令历史记录
- 命令建议

**解析优先级算法**：
1. **关键词长度优先**：长关键词优先匹配（避免误匹配）
2. **出现位置优先**：相同长度时，靠前的优先

**示例解析流程**：
```python
输入: "安装标准库 版本 1.2.3"
解析结果:
{
    'intent': 'install',
    'command': 'install',
    'args': ['标准库'],
    'kwargs': {'version': '1.2.3'},
    'confidence': 0.85,
    'original_text': '安装标准库 版本 1.2.3'
}
```

### 3. 语音反馈模块（VoiceFeedback）

**位置**：`src/package/voice/voice_feedback.py`

**核心功能**：
- 支持多种语音合成引擎（在线API/pyttsx3/eSpeak/系统TTS）
- 预定义反馈消息
- 自定义语音参数（语速、音量、语言）
- 语音保存到文件
- 统计信息追踪

**支持的TTS引擎**：
```python
class TTSEngine(Enum):
    ONLINE_API = "在线API"      # 百度/讯飞等
    LOCAL_PYTTSX3 = "本地pyttsx3"  # 离线TTS
    LOCAL_ESPEAK = "本地eSpeak"   # Linux TTS
    SYSTEM = "系统TTS"          # macOS say / Windows SAPI
```

**预定义反馈消息**：
- 成功消息：install_success, publish_success, search_success等
- 错误消息：install_failed, publish_failed等
- 提示消息：listening, processing, command_not_recognized等

## 技术实现

### 1. 语音识别降级策略

```
在线API（百度/讯飞）
    ↓ 失败
本地Whisper模型
    ↓ 失败
本地Vosk模型
    ↓ 失败
规则匹配（兜底）
```

### 2. 命令解析算法

```python
# 收集所有匹配的关键词
matched_commands = []
for command, keywords in COMMAND_KEYWORDS.items():
    for keyword in keywords:
        if keyword in text_lower:
            pos = text_lower.find(keyword)
            matched_commands.append((command, keyword, len(keyword), pos))

# 优先级排序：长度 > 位置
matched_commands.sort(key=lambda x: (-x[2], x[3]))

# 返回最佳匹配
command, keyword, _, _ = matched_commands[0]
```

**示例**：
- "列出已安装的包"：
  - 匹配："列出"(位置0, 长度2) 和 "安装"(位置2, 长度2)
  - 排序后："列出"优先（位置更靠前）
  - 结果：list命令

### 3. 包管理器集成

**新增方法**：
- `listen_voice_command()` - 监听语音命令
- `execute_voice_command()` - 执行语音命令
- `voice_interact()` - 完整语音交互流程

**使用示例**：
```python
from package import PackageManager

# 创建包管理器（启用语音）
pm = PackageManager(enable_voice=True)

# 语音交互
success, msg = pm.voice_interact()
# 流程：提示说话 → 监听 → 识别 → 解析 → 执行 → 反馈

# 或分步执行
success, text = pm.listen_voice_command()
success, msg, parsed = pm.execute_voice_command(text)
```

## 使用示例

### Python API

```python
from package import PackageManager, SpeechRecognizer, VoiceCommandParser

# ===== 方式1：完整语音交互 =====
pm = PackageManager(enable_voice=True)
success, msg = pm.voice_interact()

# ===== 方式2：分步控制 =====
# 创建语音识别器
recognizer = SpeechRecognizer(engine=SpeechEngine.LOCAL_WHISPER)

# 监听并识别
success, text = recognizer.listen_and_recognize()
if success:
    # 解析命令
    parser = VoiceCommandParser()
    result = parser.parse(text)
    
    # 执行命令
    pm = PackageManager()
    success, msg = pm.execute(result['command'], *result['args'])
    
    # 语音反馈
    feedback = VoiceFeedback()
    feedback.speak(msg)

# ===== 方式3：从音频文件识别 =====
recognizer = SpeechRecognizer()
success, text = recognizer.recognize_from_file("command.wav")
```

### 命令行使用

```bash
# 启用语音交互
zhc --voice

# 语音命令示例：
# "安装标准库"
# "搜索网络包"
# "列出已安装的包"
# "更新标准库到最新版本"
# "帮助"
```
