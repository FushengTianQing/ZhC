# 包管理器中文命令支持

## 概述

中文C语言包管理器支持**中英文命令**，用户可以使用自然的中文命令进行包管理操作。

---

## 命令映射表

| 中文命令 | 英文命令 | 功能说明 |
|:---|:---|:---|
| `安装` | `install` | 安装包及依赖 |
| `发布` | `publish` | 发布包到注册表 |
| `搜索` | `search` | 搜索包 |
| `列表` | `list` | 列出已安装包 |
| `卸载` | `uninstall` | 卸载包 |
| `更新` | `update` | 更新包 |
| `验证` | `verify` | 验证包完整性 |
| `信息` | `info` | 查看包信息 |
| `帮助` | `help` | 显示帮助信息 |
| `初始化` | `init` | 初始化新包 |

## 命令别名支持

除了完整的中英文命令，还支持以下**官方短别名**和**自定义别名**：

### 官方短别名

| 别名 | 完整命令 |
|:---|:---|
| `i` | `install` / `安装` |
| `pub` | `publish` / `发布` |
| `s` | `search` / `搜索` |
| `ls` | `list` / `列表` |
| `rm` | `uninstall` / `卸载` |
| `up` | `update` / `更新` |
| `v` | `verify` / `验证` |
| `inf` | `info` / `信息` |
| `h` | `help` / `帮助` |
| `init` | `init` / `初始化` |

### 自定义别名管理

用户可以注册、查询、删除自定义别名，配置保存在 `~/.zhc/aliases.json`。

#### 注册自定义别名

```python
from zhc.package import AliasManager

# 创建别名管理器
alias_manager = AliasManager()

# 注册别名
success, msg = alias_manager.register_alias("ins", "install")
# ✅ 别名 'ins' → 'install' 注册成功
```

#### 列出所有别名

```python
# 列出所有别名（包含官方别名和自定义别名）
all_aliases = alias_manager.list_aliases()
print(all_aliases)
# {'i': 'install', 'pub': 'publish', 'ins': 'install', ...}
```

#### 删除自定义别名

```python
# 删除别名
success, msg = alias_manager.unregister_alias("ins")
# ✅ 别名 'ins' 已删除
```

#### 获取别名信息

```python
# 获取别名详细信息
info = alias_manager.get_alias_info("i")
# {'alias': 'i', 'command': 'install', 'type': '官方别名'}
```

### 使用示例

```bash
# 使用官方短别名
zhc i 标准库        # 等同于 zhc install 标准库
zhc s 网络          # 等同于 zhc search 网络
zhc ls              # 等同于 zhc list

# 使用自定义别名（假设已注册）
zhc ins 标准库      # 等同于 zhc install 标准库
```

## 编程接口

### Python API

```python
from zhc.package import PackageManager

# 创建包管理器实例
pm = PackageManager()

# 使用中文命令
success, msg = pm.execute("安装", "标准库")
print(msg)

# 使用英文命令
success, msg = pm.execute("install", "标准库")
print(msg)

# 直接调用方法（推荐）
success, msg = pm.install("标准库")
```

### 中文C语言接口

```zhc
// 导入包管理器
导入 "包管理器"

// 创建包管理器
包管理器 pm = 创建包管理器()

// 使用中文命令
布尔型 成功 = pm.执行("安装", "标准库")

// 安装包
布尔型 成功 = pm.安装包("网络库", "1.2.0")

// 发布包
布尔型 成功 = pm.发布包("/path/to/package")

// 搜索包
字符串型[] 结果 = pm.搜索包("网络")
```

## 设计优势

1. **语言一致性**：与中文C语言语法保持一致
2. **易学易用**：中文用户更容易记忆和使用

## 技术实现

### 命令映射器

```python
class CommandMapper:
    """命令映射器"""
    CN_TO_EN = {
        "安装": "install",
        "发布": "publish",
        "搜索": "search",
        # ...
    }
    
    EN_TO_CN = {v: k for k, v in CN_TO_EN.items()}
    
    @staticmethod
    def to_english(command: str) -> str:
        return CommandMapper.CN_TO_EN.get(command, command)
    
    @staticmethod
    def to_chinese(command: str) -> str:
        return CommandMapper.EN_TO_CN.get(command, command)
```

### 执行流程

```
用户输入 → 命令识别 → 转换为内部命令 → 执行方法 → 返回结果
    ↓
"安装 标准库" 
    ↓
CommandMapper.to_english("安装") → "install"
    ↓
PackageManager.install("标准库")
    ↓
✅ 包 '标准库@1.0.0' 安装成功
```
