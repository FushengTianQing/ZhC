## 指针类型设计

### 语法

```
指针 类型名 = &变量名;
指针 空型 = 空指针值;
解引用 *指针名
```

### 示例

```
整数型 变量 = 100;
整数型* 指针 = &变量;

函数 获取值() -> 整数型 {
    返回 *指针;
}
```

### 类型映射

| 中文语法 | C语法 |
|:---|:---|
| 整数型* | int* |
| 字符型* | char* |
| 浮点型* | float* |
| 空型* | void* |

## 数组语法设计

### 一维数组

```
数组 类型名[大小];
数组 类型名[大小] = { 元素1, 元素2 };
```

### 示例

```
整数型 数组[10];
整数型 数据[5] = { 1, 2, 3, 4, 5 };

函数 获取元素(整数型 索引) -> 整数型 {
    返回 数据[索引];
}
```

### 二维数组

```
整数型 矩阵[3][4];
```

## 内存分配

### 栈分配

```
// 局部变量（自动）
整数型 局部变量 = 10;
```

### 堆分配

```
指针 整数型 ptr = 分配(sizeof(整数型));
释放(ptr);
```

### 简化语法

```
整数型* 动态数组 = 新建 整数型[大小];
删除[] 动态数组;
```

## 指针运算

### 支持的运算

| 运算 | 示例 | 说明 |
|:---|:---|:---|
| + | ptr + n | 指针偏移 |
| - | ptr - n | 指针回退 |
| - | ptr1 - ptr2 | 距离 |
| ++ | ++ptr | 递增 |
| -- | --ptr | 递减 |

### 数组与指针

```
整数型 数组[5] = {1, 2, 3, 4, 5};
整数型* ptr = 数组;

ptr[0]  // 等价于 *(ptr + 0)
ptr[2]  // 等价于 *(ptr + 2)
```

## 字符串支持

### 字符串字面量

```
字符串型 str = "你好世界";
字符型* cstr = "Hello";
```

### 字符串操作

```
函数 长度(字符型* s) -> 整数型 {
    整数型 len = 0;
    当(s[len] != '\\0') {
        len = len + 1;
    }
    返回 len;
}
```
