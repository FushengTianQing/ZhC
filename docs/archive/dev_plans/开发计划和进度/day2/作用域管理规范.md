# 作用域管理规范

## 概述

本文档定义中文C编译器第三阶段的作用域管理系统规范，包括作用域层次、可见性规则、符号查找算法等。

## 作用域体系

### 四级作用域层次

#### 1. 全局作用域 (Global Scope)
- **级别**: 0 (最外层)
- **标识符**: `global` 或空字符串
- **包含内容**:
  - 所有模块声明
  - 全局配置和宏定义
  - 跨模块共享的类型定义
- **生命周期**: 整个程序运行期间
- **可见性**: 对所有作用域可见

#### 2. 模块作用域 (Module Scope)
- **级别**: 1
- **标识符**: 模块名 (如 `数学库`)
- **包含内容**:
  - 模块内的所有符号
  - 模块级类型定义
  - 模块内部配置
- **生命周期**: 模块加载到卸载期间
- **可见性**: 根据访问控制规则决定

#### 3. 文件作用域 (File Scope)
- **级别**: 2
- **标识符**: 文件名 (如 `math_utils.c`)
- **包含内容**:
  - 文件内的静态变量和函数
  - 文件级类型定义
  - 文件内部宏定义
- **生命周期**: 文件作用域期间
- **可见性**: 仅对同一文件内的作用域可见

#### 4. 块作用域 (Block Scope)
- **级别**: 3 (最内层)
- **标识符**: 可选的块名或空
- **包含内容**:
  - 局部变量
  - 函数参数
  - 临时变量
- **生命周期**: 块执行期间
- **可见性**: 仅对当前块和嵌套块可见

### 作用域关系图
```
全局作用域 (0级)
├── 模块A (1级)
│   ├── 文件A1.c (2级)
│   │   ├── 函数块1 (3级)
│   │   └── 函数块2 (3级)
│   └── 文件A2.c (2级)
│       └── 函数块3 (3级)
└── 模块B (1级)
    └── 文件B1.c (2级)
        └── 函数块4 (3级)
```

## 可见性规则

### 访问控制级别

#### 1. 公开 (Public)
- **标识符**: `公开:` 或 `public`
- **规则**:
  - 对其他所有模块可见
  - 可以跨模块调用和引用
  - 生成公开的C符号（无static修饰）
- **使用场景**:
  - 模块API接口
  - 公共工具函数
  - 共享数据结构

#### 2. 私有 (Private)
- **标识符**: `私有:` 或 `private`
- **规则**:
  - 仅对当前模块内可见（默认）
  - 其他模块无法访问
  - 生成静态C符号（static修饰）
- **使用场景**:
  - 模块内部实现
  - 辅助函数
  - 内部状态变量

#### 3. 保护 (Protected)
- **标识符**: `保护:` 或 `protected`
- **规则**:
  - 对当前模块和子模块可见
  - 其他模块无法访问
  - 生成静态C符号（static修饰）
- **使用场景**:
  - 模块扩展点
  - 可继承的基类成员
  - 内部API（未来扩展用）

### 可见性矩阵

| 符号可见性 | 同模块内 | 子模块内 | 其他模块 | 全局 |
|------------|----------|----------|----------|------|
| 公开       | ✅       | ✅       | ✅       | ✅   |
| 保护       | ✅       | ✅       | ❌       | ❌   |
| 私有       | ✅       | ❌       | ❌       | ❌   |

### 默认可见性规则

1. **模块级符号**: 默认私有
   ```c
   // 默认私有
   整数型 内部变量 = 0;  // 等同于: 私有: 整数型 内部变量 = 0;
   ```

2. **文件级符号**: 总是私有
   ```c
   // 文件作用域符号总是私有
   静态 整数型 文件变量 = 0;
   ```

3. **块级符号**: 总是私有
   ```c
   函数 测试() -> 整数型 {
       整数型 局部变量 = 0;  // 总是私有
   }
   ```

## 符号管理

### 符号命名规则

#### 1. 原始符号名
- **格式**: 中文标识符或合法C标识符
- **示例**: `圆周率`, `加法`, `calculate_sum`

#### 2. 限定符号名 (Qualified Name)
- **格式**: `{作用域路径}::{符号名}`
- **示例**: 
  - `数学库::圆周率`
  - `数学库::工具::计算器::加法`

#### 3. C转换符号名
- **公开符号**: `{模块名}_{符号名}`
  - 示例: `数学库_圆周率`
- **私有符号**: `static {模块名}_{符号名}`
  - 示例: `static 数学库_内部变量`
- **保护符号**: `static {模块名}_{符号名}` (暂时)
  - 示例: `static 数学库_保护函数`

### 符号查找算法

#### 1. 作用域链查找
```python
def lookup_in_scope_chain(name: str, start_scope: Scope) -> Optional[SymbolInfo]:
    """
    在作用域链中查找符号
    算法: 从当前作用域开始，逐级向上查找
    """
    current = start_scope
    
    while current is not None:
        # 在当前作用域查找
        symbol = current.symbols.get(name)
        if symbol is not None:
            # 检查可见性
            if current.can_see(symbol, start_scope):
                return symbol
                
        # 向上级作用域查找
        current = current.parent
        
    return None
```

#### 2. 导入符号查找
```python
def lookup_imported_symbol(name: str, module: ModuleInfo) -> Optional[SymbolInfo]:
    """
    在导入的模块中查找符号
    算法: 检查所有导入的模块
    """
    for imported_name in module.imports:
        imported_module = get_module(imported_name)
        if imported_module is None:
            continue
            
        # 在导入模块的公开符号中查找
        if name in imported_module.public_symbols:
            return imported_module.symbols[name]
            
    return None
```

## 错误处理

### 作用域错误类型

#### 1. 作用域嵌套错误
```python
class ScopeNestingError(Exception):
    """作用域嵌套错误"""
    
    @classmethod
    def check_nesting(cls, parent: ScopeType, child: ScopeType):
        """检查作用域嵌套是否合法"""
        nesting_rules = {
            ScopeType.GLOBAL: [ScopeType.MODULE],
            ScopeType.MODULE: [ScopeType.FILE, ScopeType.BLOCK],
            ScopeType.FILE: [ScopeType.BLOCK],
            ScopeType.BLOCK: [ScopeType.BLOCK],
        }
        
        if child not in nesting_rules.get(parent, []):
            raise cls(f"非法作用域嵌套: {parent} → {child}")
```

#### 2. 符号重定义错误
```python
class SymbolRedefinitionError(Exception):
    """符号重定义错误"""
    
    @classmethod
    def check_redefinition(cls, scope: Scope, name: str):
        """检查符号是否重定义"""
        if name in scope.symbols:
            raise cls(f"符号 '{name}' 在作用域 '{scope.name}' 中已定义")
```

#### 3. 未定义符号错误
```python
class UndefinedSymbolError(Exception):
    """未定义符号错误"""
    
    @classmethod
    def check_defined(cls, symbol_info: Optional[SymbolInfo], name: str):
        """检查符号是否已定义"""
        if symbol_info is None:
            raise cls(f"符号 '{name}' 未定义")
```

### 错误恢复策略

#### 1. 作用域错误恢复
- **非法嵌套**: 自动创建中间作用域或跳过当前声明
- **作用域不匹配**: 自动插入缺失的作用域边界

#### 2. 符号错误恢复
- **符号重定义**: 重命名新符号或忽略重复定义
- **未定义引用**: 创建临时符号声明或标记为外部引用

#### 缓存优化
```python
class LookupCache:
    """查找缓存"""
    
    def __init__(self):
        self.cache: Dict[Tuple[str, Scope], SymbolInfo] = {}
        
    def get(self, name: str, scope: Scope) -> Optional[SymbolInfo]:
        """从缓存获取"""
        key = (name, id(scope))
        return self.cache.get(key)
        
    def put(self, name: str, scope: Scope, symbol: SymbolInfo):
        """放入缓存"""
        key = (name, id(scope))
        self.cache[key] = symbol
```

### 2. 内存优化

#### 作用域共享
```python
class ScopePool:
    """作用域对象池"""
    
    def __init__(self):
        self.pool: Dict[Tuple[str, ScopeType], Scope] = {}
        
    def get_scope(self, name: str, scope_type: ScopeType) -> Scope:
        """获取或创建作用域"""
        key = (name, scope_type)
        if key not in self.pool:
            self.pool[key] = Scope(name, scope_type)
        return self.pool[key]
```

#### 符号字符串驻留
```python
import sys

class StringInterner:
    """字符串驻留器"""
    
    def __init__(self):
        self.interned: Dict[str, str] = {}
        
    def intern(self, s: str) -> str:
        """驻留字符串"""
        if s not in self.interned:
            self.interned[s] = sys.intern(s)
        return self.interned[s]
```

```yaml
scope_system:
  # 作用域嵌套规则
  nesting_strictness: "strict"  # strict | lenient
```
