## 三、代码实现

### 3.1 核心功能

```python
# 方法转换
result = converter.convert_method(
    class_name="学生",
    method_code="函数 获取信息() -> 字符串型"
)
# 结果: 学生_获取信息(void *self)
```

### 3.2 转换规则

| 中文方法 | C函数 |
|:---|:---|
| 函数 名() -> 类型 | 类型 类名_名(struct 类* self) |
| 函数 构造函数(参数) -> 空型 | void 类名_constructor(struct 类* self, 参数) |
| 函数 方法(参数) -> 类型 | 类型 类名_方法(struct 类* self, 参数) |
