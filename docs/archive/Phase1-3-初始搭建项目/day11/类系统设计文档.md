## 一、概述

类系统是中文C编译器第三阶段的核心功能之一，旨在实现面向对象编程能力。本文档定义了类系统的完整语法规范、设计决策和技术实现细节。

## 二、语法规范

### 2.1 基本类定义

```c
类 类名 {
    属性:
        类型 属性名;
        类型 属性名 = 默认值;

    方法:
        函数 方法名(参数列表) -> 返回类型 {
            // 方法体
        }
}
```

### 2.2 继承语法

```c
类 派生类名 : 基类名 {
    属性:
        // 派生类属性

    方法:
        // 派生类方法
}
```

### 2.3 访问控制

中文C编译器支持三种访问控制级别：

| 关键字 | 说明 | 可见范围 |
|:---:|:---|:---|
| `公开:` | 公开 | 所有代码可访问 |
| `私有:` | 私有 | 仅类内部可访问 |
| `保护:` | 保护 | 类及子类可访问 |

### 2.4 属性声明

```c
类 学生 {
    公开:
    属性:
        字符串型 姓名;           // 公开属性
        整数型 年龄;            // 公开属性

    私有:
    属性:
        浮点型 成绩 = 0.0;     // 私有属性，带默认值
}
```

### 2.5 方法声明

```c
类 学生 {
    公开:
    方法:
        // 构造函数（公开）
        函数 构造函数(字符串型 名, 整数型 龄) -> 空型 {
            姓名 = 名;
            年龄 = 龄;
        }

        // 普通方法（公开）
        函数 获取信息() -> 字符串型 {
            返回 "学生";
        }

        // 带参数的方法
        函数 设置成绩(参数 浮点型 分) -> 空型 {
            成绩 = 分;
        }

    私有:
    方法:
        // 私有方法
        函数 内部校验() -> 逻辑型 {
            返回 成绩 >= 0;
        }
}
```

## 四、函数和方法

### 4.1 构造函数

构造函数用于初始化类的实例：

```c
函数 构造函数(字符串型 名, 整数型 龄) -> 空型 {
    姓名 = 名;
    年龄 = 龄;
}
```

**规则**：
- 名称固定为"构造函数"
- 返回类型固定为"空型"
- 可以有参数列表
- 自动在对象创建时调用

### 4.2 析构函数

析构函数用于清理资源：

```c
函数 析构函数() -> 空型 {
    // 清理资源
}
```

**规则**：
- 名称固定为"析构函数"
- 返回类型固定为"空型"
- 无参数列表
- 自动在对象销毁时调用

### 4.3 普通方法

```c
函数 方法名(参数类型 参数名, ...) -> 返回类型 {
    // 方法体
    返回 表达式;
}
```

## 五、转换规则

### 5.1 类转换为结构体

**中文类定义**：
```c
类 学生 {
    属性:
        字符串型 姓名;
        整数型 年龄;
}
```

**转换为C代码**：
```c
typedef struct Student {
    char* 姓名;
    int 年龄;
} Student;
```

### 5.2 方法转换为函数

**中文方法**：
```c
函数 设置成绩(参数 浮点型 分) -> 空型 {
    成绩 = 分;
}
```

**转换为C代码**：
```c
void Student_setScore(Student* this, float 分) {
    this->成绩 = 分;
}
```

**转换规则**：
1. 方法名添加类名前缀：`类名_方法名`
2. 第一个参数自动添加`this`指针：`类名* this`
3. 成员访问使用`this->`语法

### 5.3 构造函数转换

**中文构造函数**：
```c
函数 构造函数(字符串型 名, 整数型 龄) -> 空型 {
    姓名 = 名;
    年龄 = 龄;
}
```

**转换为C代码**：
```c
Student* Student_constructor(Student* this, char* 名, int 龄) {
    this->姓名 = 名;
    this->年龄 = 龄;
    return this;
}
```

### 5.4 继承转换

**中文继承类**：
```c
类 大学生 : 学生 {
    属性:
        字符串型 专业;

    方法:
        函数 获取专业() -> 字符串型 {
            返回 专业;
        }
}
```

**转换为C代码**：
```c
typedef struct Undergraduate {
    Student base;    // 基类
    char* 专业;       // 派生属性
} Undergraduate;

char* Undergraduate_getMajor(Undergraduate* this) {
    return this->专业;
}
```

## 六、继承机制

### 6.1 单继承

中文C编译器支持单继承，即一个类只能有一个基类：

```c
// 合法的继承
类 大学生 : 学生 { }

// 非法的继承（不支持多继承）
类 研究生 : 学生, 教师 { }  // 错误
```

### 6.2 继承链

继承可以形成链状结构：

```c
类 人类 { }
类 学生 : 人类 { }
类 大学生 : 学生 { }
```

### 6.3 成员访问

| 成员类型 | 基类内部 | 派生类内部 | 外部 |
|:---|:---:|:---:|:---:|
| 公开成员 | ✅ | ✅ | ✅ |
| 私有成员 | ✅ | ❌ | ❌ |
| 保护成员 | ✅ | ✅ | ❌ |

## 七、完整示例

### 7.1 学生类

```c
类 学生 {
    公开:
    属性:
        字符串型 姓名;
        整数型 年龄;

    私有:
    属性:
        浮点型 成绩 = 0.0;

    方法:
        函数 构造函数(字符串型 名, 整数型 龄) -> 空型 {
            姓名 = 名;
            年龄 = 龄;
        }

        函数 获取信息() -> 字符串型 {
            返回 姓名;
        }

        函数 设置成绩(参数 浮点型 分) -> 空型 {
            成绩 = 分;
        }

        函数 获取成绩() -> 浮点型 {
            返回 成绩;
        }
}
```

### 7.2 派生大学生类

```c
类 大学生 : 学生 {
    公开:
    属性:
        字符串型 专业;
        字符串型 学号;

    方法:
        函数 构造函数(字符串型 名, 整数型 龄, 字符串型 专, 字符串型 号) -> 空型 {
            姓名 = 名;
            年龄 = 龄;
            专业 = 专;
            学号 = 号;
        }

        函数 获取专业() -> 字符串型 {
            返回 专业;
        }
}
```

### 7.3 使用示例

```c
模块 主程序 {
    导入 学生
    导入 大学生

    主函数() -> 整数型 {
        大学生 小明 = 新 大学生("小明", 20, "计算机", "20230001");

        字符串型 信息 = 小明->获取信息();
        字符串型 专业 = 小明->获取专业();

        打印("学生: %s\n", 信息);
        打印("专业: %s\n", 专业);

        删除 小明;

        返回 0;
    }
}
```

### 8.2 ClassInfo 类

```python
@dataclass
class ClassInfo:
    name: str                           # 类名
    base_class: Optional[str] = None     # 基类名
    attributes: List[AttributeInfo]      # 属性列表
    methods: List[MethodInfo]          # 方法列表
    visibility: Visibility             # 类可见性

    def get_public_attributes(self) -> List[AttributeInfo]
    def get_public_methods(self) -> List[MethodInfo]
    def get_constructor(self) -> Optional[MethodInfo]
```

### 8.3 AttributeInfo 类

```python
@dataclass
class AttributeInfo:
    name: str                    # 属性名
    type_name: str               # 类型名
    visibility: Visibility       # 可见性
    default_value: Optional[str]  # 默认值
```

### 8.4 MethodInfo 类

```python
@dataclass
class MethodInfo:
    name: str                    # 方法名
    return_type: str            # 返回类型
    parameters: List[Tuple[str, str]]  # 参数列表
    visibility: Visibility       # 可见性
    is_constructor: bool         # 是否为构造函数
    is_destructor: bool         # 是否为析构函数
```
