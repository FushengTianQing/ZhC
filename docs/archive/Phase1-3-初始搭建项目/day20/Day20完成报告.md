## 核心功能

### 1. 内存语法解析

| 语法 | C代码 |
|:---|:---|
| `新建 类型 ptr;` | `type* ptr = malloc(sizeof(type));` |
| `删除 ptr;` | `free(ptr);` |
| `分配(sizeof(Type))` | `malloc(sizeof(Type))` |
| `释放(ptr);` | `free(ptr);` |

### 2. 智能指针

| 类型 | 语法 | C++生成 |
|:---|:---|:---|
| 独享指针 | `独享指针<Type> var;` | `std::unique_ptr<Type>` |
| 共享指针 | `共享指针<Type> var;` | `std::shared_ptr<Type>` |

### 3. 内存安全检测

| 检测类型 | 说明 |
|:---|:---|
| 内存泄漏 | 检测未释放的malloc |
| 双重释放 | 检测重复free |
| 空指针解引用 | 检查NULL指针使用 |
| 未初始化使用 | 检测未分配就使用 |

## 语法规则

### 新建语法
```
新建 类型 变量名;
新建 类型 变量名 = 初值;
新建 类型 数组[大小];
```

### 删除语法
```
删除 变量名;
删除数组 数组名;
```

### 智能指针语法
```
独享指针<类型> 变量名;
共享指针<类型> 变量名;
```
