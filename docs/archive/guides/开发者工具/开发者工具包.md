# 中文C编译器 - 开发者工具包

## 概述

本工具包包含编译器辅助开发、调试和性能优化的实用工具。

## 工具列表

### 1. 错误处理工具

位于: `src/phase3/core/day27/tool_chain.py`

```python
from phase3.core.day27.tool_chain import EnhancedErrorHandler, ErrorCode

handler = EnhancedErrorHandler()
handler.add_error(ErrorCode.MEMORY_LEAK, 25, suggestion="释放内存")
print(handler.format_summary())
```

### 2. 性能分析工具

```python
from phase3.core.day27.tool_chain import PerformanceOptimizer

optimizer = PerformanceOptimizer()
optimizer.measure('lexical_time', some_function)
print(optimizer.get_report())
```

### 3. 调试信息工具

```python
from phase3.core.day27.tool_chain import DebugInfoGenerator

debugger = DebugInfoGenerator()
debugger.add_symbol('ptr', 'int*', 10)
print(debugger.get_symbol_info('ptr'))
```

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行测试

```bash
# 运行所有测试
bash tests/run_tests.sh

# 运行特定测试套件
python3 tests/test_suite7/test_module_system.py
python3 tests/test_suite8/test_class_system_v2.py
python3 tests/test_suite9/test_suite9_complete.py
```

### 编译示例

```bash
python3 src/zhc.py examples/hello.zhc
```

## 调试模式

启用调试输出：

```python
import os
os.environ['ZHC_DEBUG'] = '1'
```

## 性能分析

```bash
# 生成性能报告
python3 src/phase3/core/day9/performance_analyzer.py
```

## 常用命令
