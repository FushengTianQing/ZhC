### 3.3 语法分析器

#### 声明解析

**函数声明**：
```zhc
函数 add(整数型 a, 整数型 b) 整数型 {
    返回 a + b;
}
```

解析流程：
1. 消耗 '函数' 关键字
2. 解析函数名
3. 解析参数列表
4. 解析返回类型
5. 解析函数体

**变量声明**：
```zhc
整数型 count = 0;
浮点型 data[100];
常量 整数型 MAX = 100;
```

#### 语句解析

**如果语句**：
```zhc
如果 (count < 10) {
    count += 1;
} 否则 {
    count -= 1;
}
```

**循环语句**：
```zhc
循环 (整数型 i = 0; i < 10; i += 1) {
    count += 1;
}
```

**当循环**：
```zhc
当 (count < 10) {
    count += 1;
}
```

#### 表达式解析

**运算符优先级**：
```python
# 从低到高
assignment  →  =, +=, -=, ...
or          →  ||
and         →  &&
equality    →  ==, !=
comparison  →  <, <=, >, >=
addition    →  +, -
multiplication → *, /, %
unary       →  -, !, ~, ++, --
postfix     →  (), [], ., ->, ++, --
```

**解析方法**：
```python
def parse_expression(self) -> ASTNode:
    return self.parse_assignment()

def parse_assignment(self) -> ASTNode:
    expr = self.parse_or()
    if self.match(TokenType.ASSIGN, ...):
        operator = self.advance().value
        value = self.parse_assignment()
        return AssignExprNode(expr, value, operator)
    return expr

# 其他parse方法...
```

### 5.2 错误恢复

**同步机制**：
```python
def synchronize(self):
    """错误恢复：跳到下一个语句"""
    while not self.is_at_end():
        # 在分号处恢复
        if self.current_token().type == TokenType.SEMICOLON:
            self.advance()
            return
        
        # 在关键字处恢复
        if self.current_token().type in (
            TokenType.FUNCTION, TokenType.STRUCT,
            TokenType.IF, TokenType.WHILE, TokenType.FOR,
            TokenType.RETURN, ...
        ):
            return
        
        self.advance()
```

## 六、使用示例

### 6.1 词法分析

```python
from zhpp.parser import Lexer

code = """
整数型 主函数() {
    整数型 计数 = 0;
    返回 计数;
}
"""

lexer = Lexer(code)
tokens = lexer.tokenize()

for token in tokens:
    print(token)

# 输出:
# Token(INT, '整数型', 2:1)
# Token(IDENTIFIER, '主函数', 2:5)
# Token(LPAREN, '(', 2:8)
# Token(RPAREN, ')', 2:9)
# Token(LBRACE, '{', 2:11)
# ...
```

### 6.2 语法分析

```python
from zhpp.parser import parse, ASTPrinter

code = """
整数型 add(整数型 a, 整数型 b) {
    返回 a + b;
}
"""

ast, errors = parse(code)

# 打印AST
printer = ASTPrinter()
ast.accept(printer)

# 输出:
# Program
#   FunctionDecl: add
#     ReturnType:
#       PrimitiveType: 整数型
#     Params:
#       ParamDecl: a
#       ParamDecl: b
#     Body:
#       BlockStmt
#         ReturnStmt
#           BinaryExpr: +
#             Identifier: a
#             Identifier: b
```

### 6.3 错误处理

```python
from zhpp.parser import parse

code = """
整数型 主函数( {
    返回 0;
}
"""

ast, errors = parse(code)

if errors:
    print("发现错误:")
    for error in errors:
        print(f"  ✗ {error}")

# 输出:
# 发现错误:
#   ✗ 第2行第16列: 期望 ')', 但找到 '{'
```

## 八、已知限制

### 8.1 当前限制

1. **函数声明解析不完善**
   - 参数列表解析有误
   - 返回类型位置不规范

2. **类型系统简单**
   - 未支持函数类型
   - 未支持复杂类型组合

3. **错误恢复粗糙**
   - 仅在分号和关键字处恢复
   - 错误信息不够友好

### 8.2 改进方向

1. **完善函数声明解析**
