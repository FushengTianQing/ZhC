## 问题分析

### 原有问题
- 即使只修改一个函数,也需要重新编译整个文件
- 依赖该函数的其他函数无法智能判断是否需要重编译
- 缓存粒度太粗,无法精准复用

### 性能瓶颈
1. **函数级缓存缺失** - 只在文件级别缓存
2. **依赖追踪不足** - 无法判断函数变更影响范围
3. **增量编译不精细** - 修改一个函数导致全部重编译

## 优化方案

### 核心设计

#### 1. 函数哈希计算

```python
class FunctionLevelCache:
    """函数级编译缓存"""
    
    def compute_function_hash(
        self,
        func_name: str,
        func_body: str,
        params: str = "",
        return_type: str = "",
        symbols_used: Set[str] = None
    ) -> FunctionHash:
        """计算函数的完整哈希"""
        # 内容哈希
        content_str = f"{func_name}|{params}|{return_type}|{func_body}"
        content_hash = hashlib.sha256(content_str.encode()).hexdigest()
        
        # 依赖哈希
        dep_str = "|".join(sorted(symbols_used or set()))
        dependency_hash = hashlib.sha256(dep_str.encode()).hexdigest()
        
        # 完整哈希
        full_hash = hashlib.sha256(f"{content_hash}|{dependency_hash}".encode()).hexdigest()
        
        return FunctionHash(...)
```

#### 2. 缓存存取

```python
    def get_or_compile(
        self,
        func_name: str,
        func_body: str,
        params: str,
        return_type: str,
        symbols_used: Set[str],
        compiler_func,
        dependencies: Set[str] = None
    ) -> Tuple[str, bool, float]:
        """获取缓存或重新编译"""
        
        # 尝试获取缓存
        cached, status = self.get(func_name, func_body, params, return_type, symbols_used)
        
        if status == CacheStatus.HIT:
            # 缓存命中，直接返回
            return cached.compiled_code, True, 0.0
        
        # 重新编译
        compiled_code = compiler_func(func_body)
        
        # 存入缓存
        self.put(...)
        
        return compiled_code, False, compile_time
```

#### 3. 依赖追踪

```python
    def invalidate(self, func_name: str):
        """使函数缓存失效"""
        # 删除缓存
        
        # 使依赖此函数的函数也失效
        dependents = [
            name for name, deps in self._dependency_graph.items()
            if func_name in deps
        ]
        
        for dependent in dependents:
            self.invalidate(dependent)
```

### 与增量编译器集成

```python
class IncrementalCompiler:
    """增量编译器"""
    
    def __init__(self):
        self.function_cache = FunctionLevelCache()
    
    def compile_file(self, filepath):
        """编译文件"""
        ast = self.parser.parse_file(filepath)
        
        for func in ast.functions:
            # 使用函数级缓存
            compiled, used_cache, time_ms = self.function_cache.get_or_compile(
                func_name=func.name,
                func_body=func.body,
                params=func.params,
                return_type=func.return_type,
                symbols_used=self._extract_symbols(func),
                compiler_func=self.compile_function,
                dependencies=self._extract_dependencies(func),
            )
            
            if not used_cache:
                print(f"编译函数 {func.name}: {time_ms:.2f}ms")
```
