## 📝 使用示例

### 1. 创建主函数
输入 `main` + `Tab`：
```zhc
整数型 主函数() {
    // 光标位置
    返回 0;
}
```

### 2. 创建类
输入 `class` + `Tab`：
```zhc
类 类名 {
    公开:
        // 公开成员

    私有:
        // 私有成员
}
```

### 3. 创建函数
输入 `func` + `Tab`：
```zhc
函数 函数名(参数) -> 返回类型 {
    // 实现
    返回 结果;
}
```

## 🔧 技术实现

### 1. 语法高亮
使用 TextMate 语法（`.tmLanguage.json`）：
- 正则表达式匹配语法元素
- 嵌套规则支持复杂结构
- Scope命名遵循VS Code规范

### 2. 代码补全
使用 VS Code API：
- `vscode.CompletionItem` 提供补全项
- `vscode.SnippetString` 支持代码片段
- 上下文感知的补全建议

### 3. 悬停提示
使用 `vscode.Hover` API：
- 识别当前单词
- 返回对应文档
- Markdown格式支持

### 4. 编译运行
使用 Node.js `child_process`：
- 异步执行编译器
- 捕获输出和错误
- 终端集成运行
