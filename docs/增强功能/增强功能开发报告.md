## 二、函数内联优化器（任务9）

### 2.1 核心功能

**文件**: `src/opt/function_inline.py`  
**代码量**: ~500行  
**测试覆盖**: 5个单元测试，全部通过  

#### 功能特性

1. **内联决策**：基于启发式规则判断是否内联
   - 小函数优先（指令数 <= 阈值）
   - 频繁调用的函数优先
   - 避免内联递归函数
   - 避免内联复杂控制流

2. **函数体复制**：正确处理参数替换和返回值
   - 参数替换映射
   - 返回值处理
   - 递归检测

3. **成本估算**：基于函数大小和调用频率的优化决策
   - 收益/成本比计算
   - 智能内联策略

#### 核心类

- **FunctionInliner**: 函数内联器主类
- **FunctionInfo**: 函数信息（参数、函数体、返回类型）
- **CallSite**: 调用点信息
- **InlineDecision**: 内联决策枚举

#### 使用示例

```python
from opt.function_inline import FunctionInliner

# 创建内联器
inliner = FunctionInliner(max_instruction_count=20)

# 注册函数
inliner.register_function(
    name="add",
    params=["a", "b"],
    body="return a + b;",
    return_type="int"
)

# 内联调用
result = inliner.inline_call("add", ["10", "20"])
print(result)  # "10 + 20"
```

## 三、循环优化器（任务10）

### 3.1 核心功能

**文件**: `src/opt/loop_optimizer.py`  
**代码量**: ~480行  
**测试覆盖**: 4个单元测试，全部通过  

#### 功能特性

1. **循环展开**（Loop Unrolling）
   - 减少循环开销
   - 完全展开或部分展开
   - 支持动态迭代次数

2. **循环合并**（Loop Fusion）
   - 合并相邻循环
   - 检查迭代范围相同
   - 避免数据依赖冲突

3. **循环不变量外提**（Loop Invariant Code Motion）
   - 将不变代码移到循环前
   - 减少循环内计算

4. **强度削减**（Strength Reduction）
   - 乘除法替换为移位操作
   - `i * 2` → `i << 1`
   - `i / 4` → `i >> 2`

#### 核心类

- **LoopOptimizer**: 循环优化器主类
- **LoopInfo**: 循环信息（循环变量、范围、体）
- **OptimizationType**: 优化类型枚举

#### 使用示例

```python
from opt.loop_optimizer import LoopOptimizer

# 创建优化器
optimizer = LoopOptimizer(max_unroll_iterations=10)

# 优化代码
code = """
循环 i 从 0 到 5 {
    整数型 x = i * 2;
}
"""

optimized = optimizer.optimize(code)
print(optimized)
```

#### 使用示例

# 创建后端

    optimization_level="O2",
    enable_simd=True
)

    "output",
    "app",
    export_functions=["main", "add"]

## 六、Sanitizers支持（任务13）

### 6.1 核心功能

**文件**: `src/tool/sanitizers.py`  
**代码量**: ~450行  
**测试覆盖**: 4个单元测试，全部通过  

#### 功能特性

1. **AddressSanitizer (ASan)**：内存错误检测
   - 堆缓冲区溢出
   - 栈缓冲区溢出
   - Use-after-free
   - 内存泄漏

2. **MemorySanitizer (MSan)**：未初始化内存读取检测
   - 检测未初始化值的使用
   - 避免潜在的安全问题

3. **ThreadSanitizer (TSan)**：数据竞争检测
   - 检测数据竞争
   - 死锁检测

4. **UndefinedBehaviorSanitizer (UBSan)**：未定义行为检测
   - 整数溢出
   - 除零错误
   - 空指针解引用

5. **LeakSanitizer (LSan)**：内存泄漏检测
   - 检测内存泄漏
   - 生成泄漏报告

#### 核心类

- **SanitizerIntegration**: Sanitizers集成类
- **SanitizerType**: Sanitizer类型枚举
- **SanitizerResult**: 检测结果

#### 使用示例

```python
from tool.sanitizers import SanitizerIntegration

# 创建sanitizer
sanitizer = SanitizerIntegration()

# 使用AddressSanitizer测试
result = sanitizer.test_with_asan("main.c")

if result.passed:
    print("内存检测通过")
else:
    print(f"检测到{len(result.errors)}个错误")
    for error in result.errors:
        print(f"  - {error}")
```
