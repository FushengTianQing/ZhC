## 一、快速开始

### 1.1 基本语法

```c
// 定义一个模块
模块 数学库 {
    公开:
        函数 加法(整数型 a, 整数型 b) -> 整数型 {
            返回 a + b;
        }

        函数 乘法(整数型 a, 整数型 b) -> 整数型 {
            返回 a * b;
        }

    私有:
        整数型 版本号 = 100;
}

// 使用模块
模块 主程序 {
    导入 数学库

    主函数() -> 整数型 {
        整数型 结果 = 加法(3, 4);
        返回 结果;
    }
}
```

## 三、导入导出

### 3.1 基本导入
```c
导入 数学库
导入 工具库
```

### 3.2 带别名导入
```c
导入 数学库 为 M
M.加法(1, 2)  // 使用别名调用
```

### 3.3 可见性控制
```c
模块 测试库 {
    公开:           // 公开符号，对其他模块可见
        函数 公开函数();

    私有:           // 私有符号，仅当前模块可见
        整数型 内部变量;

    保护:           // 保护符号，当前和子模块可见
        整数型 保护数据;
}
```

## 七、完整示例

### 7.1 多模块项目
```c
// math.zhc
模块 数学库 {
    公开:
        函数 加法(整数型 a, 整数型 b) -> 整数型 {
            返回 a + b;
        }

        函数 乘法(整数型 a, 整数型 b) -> 整数型 {
            返回 a * b;
        }
}
```

```c
// main.zhc
模块 主程序 {
    导入 数学库

    主函数() -> 整数型 {
        整数型 a = 10;
        整数型 b = 20;

        整数型 和 = 数学库.加法(a, b);
        整数型 积 = 数学库.乘法(a, b);

        返回 0;
    }
}
```

### 7.2 使用别名的示例
```c
模块 主程序 {
    导入 数学库 为 M
    导入 工具库 为 U

    主函数() -> 整数型 {
        整数型 结果 = M.加法(1, 2);
        U.打印结果(结果);
        返回 0;
    }
}
```

## 九、常见问题

### Q1: 如何定义私有函数？
```c
模块 我的模块 {
    私有:
        函数 内部帮助() -> 整数型 {
            返回 42;
        }
}
```

### Q2: 如何导入多个模块？
```c
导入 模块A
导入 模块B
导入 模块C
```

### Q3: 版本号格式是什么？
版本号格式：`主版本.次版本.修订号`（如 `1.2.3`）

### Q4: 条件编译符号在哪里定义？
在编译器的条件编译处理器中定义：
```python
compiler = ConditionalCompiler()
compiler.define_symbol("调试模式")
```
