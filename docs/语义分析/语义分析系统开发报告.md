## 二、核心模块详解

### 2.1 类型检查器（TypeChecker）

**文件**: `src/analyzer/type_checker.py`  
**代码量**: ~480行  
**测试**: 8个单元测试，100%通过

#### 核心功能

1. **类型系统**（TypeInfo）
   - 7种基本类型：整数型、短整型、长整型、浮点型、双精度型、字符型、空型
   - 4种复合类型：指针、数组、函数、结构体
   - 类型属性：const、volatile、signed

2. **类型检查**
   - `check_type_decl()`: 类型声明检查
   - `check_assignment()`: 赋值类型兼容性检查
   - `check_binary_op()`: 二元运算类型检查
   - `check_unary_op()`: 一元运算类型检查
   - `check_function_call()`: 函数调用类型检查

3. **类型创建**
   - `create_pointer_type()`: 创建指针类型
   - `create_array_type()`: 创建数组类型
   - `create_function_type()`: 创建函数类型

#### 技术特点

- ✅ 支持隐式类型转换
- ✅ 自动检测精度丢失
- ✅ 指针类型推导
- ✅ 数组转指针支持

### 2.2 作用域检查器（ScopeChecker）

**文件**: `src/analyzer/scope_checker.py`  
**代码量**: ~420行  
**测试**: 6个单元测试，100%通过

#### 核心功能

1. **作用域管理**（Scope）
   - 全局作用域（level 0）
   - 函数作用域（level 1）
   - 代码块作用域（level 2+）
   - 作用域栈管理

2. **符号管理**（Symbol）
   - 6种符号类别：变量、函数、参数、类型定义、结构体、标签
   - 符号属性：全局、静态、常量、已初始化、已使用

3. **检查功能**
   - `declare_variable()`: 变量声明检查
   - `declare_function()`: 函数声明检查
   - `lookup_symbol()`: 符号查找
   - `check_assignable()`: 可赋值性检查
   - `check_initialized()`: 初始化检查

#### 技术特点

- ✅ 变量遮蔽检测
- ✅ 未使用变量警告
- ✅ 未初始化变量检测
- ✅ 常量赋值检查

### 2.3 函数重载解析器（OverloadResolver）

**文件**: `src/analyzer/overload_resolver.py`  
**代码量**: ~230行  
**测试**: 3个单元测试，100%通过

#### 核心功能

1. **重载解析**
   - `resolve()`: 解析最佳匹配函数
   - `_compute_match_score()`: 计算参数匹配分数
   - `_select_best_match()`: 选择最佳候选

2. **匹配策略**
   - 完全匹配（100分）
   - 数值类型转换（50-90分）
   - 指针类型转换（70-95分）
   - 数组转指针（85分）

3. **歧义检测**
   - 检测多个同等匹配候选
   - 生成重载歧义错误

#### 技术特点

- ✅ 智能匹配分数计算
- ✅ 歧义检测与报告
- ✅ 隐式转换警告

## 五、使用示例

### 5.1 基本使用

```python
from zhc.analyzer import SemanticAnalyzer

# 创建分析器
analyzer = SemanticAnalyzer()

# 分析变量声明
result = analyzer.analyze_variable_decl(1, "整数型", "count")
if result.success:
    print("变量声明有效")

# 分析函数声明
int_type = analyzer.get_type("整数型")
result = analyzer.analyze_function_decl(
    2, "add", int_type, [("a", int_type), ("b", int_type)]
)

# 生成报告
print(analyzer.report())
```

### 5.2 错误检测

```python
# 类型错误检测
result = analyzer.analyze_variable_decl(1, "未知类型", "x")
# 错误: 未知类型 '未知类型'

# 作用域错误检测
analyzer.analyze_variable_decl(1, "整数型", "x")
analyzer.analyze_variable_decl(2, "整数型", "x")
# 错误: 变量 'x' 重复声明

# 类型不匹配检测
int_type = analyzer.get_type("整数型")
void_type = analyzer.get_type("空型")
analyzer.analyze_assignment(3, "x", void_type)
# 错误: 无法将类型 '空型' 赋值给类型 '整数型'
```

## 六、性能分析

### 6.1 复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|:---|:---|:---|
| 类型检查 | O(1) | O(1) |
| 符号查找 | O(n) | O(n) |
| 重载解析 | O(m*k) | O(m) |

n: 作用域嵌套深度  
m: 候选函数数量  
k: 参数数量

### 6.2 内存占用

- TypeInfo: ~64字节/类型
- Symbol: ~128字节/符号
- Scope: ~256字节/作用域
