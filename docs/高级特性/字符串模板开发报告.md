## 三、模板语法详解

### 3.1 变量插值

**基础语法**：
```
{变量名}
```

**示例**：
```zhc
字符串型 模板 = "你好，{名字}！";
字符串型 结果 = 渲染模板(模板, {"名字": "张三"});
// 输出: "你好，张三！"
```

### 3.2 嵌套访问

**语法**：
```
{对象.属性}
{用户.名字}
{用户.地址.城市}
```

**示例**：
```zhc
结构体 用户信息 {
    字符串型 名字;
    整数型 年龄;
};

用户信息 用户 = {"李四", 25};
字符串型 模板 = "用户：{用户.名字}，年龄：{用户.年龄}岁";
字符串型 结果 = 渲染模板(模板, {用户});
// 输出: "用户：李四，年龄：25岁"
```

### 3.3 条件渲染

**语法**：
```
{% if 条件 %}
    内容
{% endif %}
```

**示例**：
```zhc
字符串型 模板 = """
你好，{用户.名字}！
{% if 用户.年龄 >= 18 %}
你是成年人。
{% endif %}
""";
```

### 3.4 循环渲染

**语法**：
```
{% for 变量 in 集合 %}
    内容：{变量}
{% endfor %}
```

**示例**：
```zhc
字符串型 模板 = """
商品列表：
{% for 商品 in 商品列表 %}
- {商品.名称}：{商品.价格}元
{% endfor %}
""";
```

### 3.5 默认值

**语法**：
```
{变量名|默认值}
```

**示例**：
```zhc
字符串型 模板 = "用户名：{用户名|匿名}";
字符串型 结果 = 渲染模板(模板, {});
// 输出: "用户名：匿名"
```

### 3.6 表达式

**语法**：
```
{{ 表达式 }}
```

**示例**：
```zhc
字符串型 模板 = "总价：{{ 单价 * 数量 }}元";
```

### 3.7 注释

**语法**：
```
{# 注释内容 #}
```

**示例**：
```zhc
字符串型 模板 = """
你好，{名字}！
{# 这是一段注释，不会出现在渲染结果中 #}
""";
```

## 四、核心类详解

### 4.1 TemplateVariable

**功能**：表示模板中的变量引用

**属性**：
- `name`: 变量名
- `path`: 访问路径（如 `["用户", "名字"]`）
- `default_value`: 默认值

**方法**：
- `to_c_code()`: 生成C代码

**示例**：
```python
var = TemplateVariable(
    name='用户.年龄',
    path=['用户', '年龄'],
    default_value='未知'
)
c_code = var.to_c_code()
# 输出: '(用户.年龄 ? 用户.年龄 : "未知")'
```

### 4.2 TemplateBlock

**功能**：表示模板中的一个块

**属性**：
- `block_type`: 块类型（文本/变量/条件/循环）
- `content`: 内容
- `children`: 子块列表
- `condition`: 条件表达式（条件块）
- `variable`: 循环变量（循环块）
- `collection`: 循环集合（循环块）

**方法**：
- `to_c_code()`: 生成C代码

### 4.3 TemplateCache

**功能**：缓存已解析的模板

**方法**：
- `get(template_id)`: 获取缓存的模板
- `set(template_id, blocks)`: 缓存模板
- `clear()`: 清空缓存
- `get_statistics()`: 获取缓存统计

### 4.4 TemplateEngine

**功能**：模板引擎主类

**方法**：
- `parse(template)`: 解析模板字符串
- `compile(template, function_name)`: 编译模板为C函数
- `render(template, context)`: 渲染模板（Python实现）
- `get_statistics(template)`: 获取模板统计

```zhc
║ 年龄：{用户.年龄}岁            ║

║ 状态：成年人                    ║

║ 状态：未成年人                  ║

```

### 5.3 编译模板

```zhc
// 编译为独立函数

字符串型 渲染用户卡片(字典型 上下文) {
    字符串型 result = zhc_malloc(1024);
    字符串型 buffer = zhc_malloc(256);
    result[0] = '\0';
    
    zhc_strcat(result, "╔════════════════════════════════╗\n");
    // ... 更多代码
    
    zhc_free(buffer);
    return result;
}
*/

// 后续使用（直接调用函数）
字符串型 卡片 = 渲染用户卡片(用户数据);
```
