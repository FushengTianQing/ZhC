## 功能概述

为中文C语言包管理器添加了完整的别名系统，支持**官方短别名**和**用户自定义别名**，提升命令输入效率。

---

## 核心设计

### 1. AliasManager 别名管理器

**位置**：`src/zhpp/package/pkg_manager.py`

**核心功能**：
- 管理官方短别名（10个）
- 支持自定义别名注册/注销
- 配置持久化（`~/.zhc/aliases.json`）
- 别名信息查询

**预定义的官方短别名**：
```python
SHORT_ALIASES = {
    "i": "install",
    "pub": "publish",
    "s": "search",
    "ls": "list",
    "rm": "uninstall",
    "up": "update",
    "v": "verify",
    "inf": "info",
    "h": "help",
    "init": "init",
}
```

### 2. CommandMapper 命令映射器增强

**新增方法**：
- `normalize(command)` - 三层解析（别名→中文→英文）
- `is_alias(command)` - 判断是否为别名
- `get_alias_manager()` - 获取别名管理器实例（懒加载）

**解析优先级**：
```
用户输入 → 别名解析 → 中文转换 → 英文验证 → 标准命令
```

---

## 技术实现

### 1. 别名解析流程

```python
# 示例：用户输入 "i"
CommandMapper.normalize("i")
    ↓
AliasManager.resolve("i")
    ↓
返回 "install"
    ↓
PackageManager.execute("install", ...)
```

### 2. 配置文件格式

**文件路径**：`~/.zhc/aliases.json`

```json
{
  "aliases": {
    "ins": "install",
    "sch": "search",
    "myalias": "install"
  }
}
```

### 3. 别名验证规则

- **不能为空**：别名和命令都不能为空字符串
- **不能冲突**：别名不能与标准命令（中英文）冲突
- **必须有效**：别名指向的命令必须是有效命令
- **官方保护**：官方短别名不能删除

## 使用示例

### Python API

```python
from zhc.package import PackageManager, AliasManager, CommandMapper

# 创建包管理器
pm = PackageManager()

# 使用短别名
success, msg = pm.execute("i", "标准库")  # i → install
print(msg)
# ✅ 包 '标准库@1.0.0' 安装成功

# 使用中文命令
success, msg = pm.execute("安装", "标准库")
# ✅ 包 '标准库@1.0.0' 安装成功

# 使用自定义别名
alias_manager = AliasManager()
alias_manager.register_alias("ins", "install")
success, msg = pm.execute("ins", "标准库")
# ✅ 包 '标准库@1.0.0' 安装成功
```

### 命令行使用

```bash
# 官方短别名
zhc i 标准库        # 安装包
zhc s 网络          # 搜索包
zhc ls              # 列出已安装

# 中文命令
zhc 安装 标准库
zhc 搜索 网络
zhc 列表

# 英文命令
zhc install 标准库
zhc search 网络
zhc list
```

## 架构优势

### 1. 三层解析架构

```
用户输入
    ↓
【别名层】解析别名（自定义 > 官方）
    ↓
【中文层】中文命令转英文
    ↓
【英文层】英文命令验证
    ↓
标准命令执行
```

### 2. 懒加载设计

```python
@classmethod
def get_alias_manager(cls) -> AliasManager:
    """获取别名管理器实例（懒加载）"""
    if cls._alias_manager is None:
        cls._alias_manager = AliasManager()
    return cls._alias_manager
```

**优点**：
- 减少启动开销
- 避免重复初始化
- 类级别共享实例

### 3. 配置持久化

- 自动加载配置文件
- 修改即时保存
- 配置文件损坏时自动忽略
