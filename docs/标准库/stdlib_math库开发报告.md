## 附录：使用示例

### A.1 stdlib.h 示例

```zhc
导入 stdlib

函数 主函数() -> 整数型 {
    // 内存管理
    整数型指针 arr = (整数型指针)分配内存(10 * sizeof(整数型));
    
    // 类型转换
    整数型 num = 字符串转整数("123");
    字符型 缓冲区[20];
    整数转字符串(num, 缓冲区, 20);
    
    // 随机数
    初始化随机种子(时间(NULL));
    整数型 dice = 随机范围(1, 6);
    
    释放内存(arr);
    返回 0;
}
```

### A.2 math.h 示例

```zhc
导入 math

函数 主函数() -> 整数型 {
    // 三角函数
    浮点型 s = 正弦(圆周率 / 6);   // sin(30°)
    浮点型 c = 余弦(圆周率 / 3);   // cos(60°)
    
    // 指数对数
    浮点型 e = 自然指数(1.0);      // e^1
    浮点型 ln = 自然对数(自然常数); // ln(e) = 1
    
    // 幂运算
    浮点型 p = 幂(2.0, 10.0);      // 2^10 = 1024
    浮点型 r = 平方根(16.0);       // √16 = 4
    
    // 取整
    浮点型 up = 向上取整(3.2);     // 4.0
    浮点型 down = 向下取整(3.8);   // 3.0
    
    返回 0;
}
```
