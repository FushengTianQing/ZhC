## 5.5 LLM API 对接能力

### 5.5.1 支持的模型

| 模型 | 适配器 | 能力评分 |
|:---|:---|:---:|
| **OpenAI GPT-4o** | `AdapterOpenAI` | 代码生成: 95, 错误修复: 90, 中文: 85 |
| **Anthropic Claude 3.5** | `AdapterAnthropic` | 代码生成: 93, 错误修复: 95, 中文: 80 |
| **Google Gemini 2.0** | `AdapterGoogle` | 代码生成: 88, 错误修复: 85, 中文: 88 |
| **本地模型 (Ollama/LM Studio)** | `AdapterLocal` | 可控、成本低、隐私保护 |
| **DeepSeek** | `AdapterDeepSeek` | 中文优化: 95, 代码: 88 |

### 5.5.2 API 配置

```bash
# OpenAI
export OPENAI_API_KEY=sk-...

# Anthropic
export ANTHROPIC_API_KEY=sk-ant-...

# Google
export GOOGLE_API_KEY=...

# 本地模型
export OLLAMA_HOST=http://localhost:11434
export LM_STUDIO_HOST=http://localhost:1234

# ZhC 自动检测并使用
zhc -ai compile.zhc
```
