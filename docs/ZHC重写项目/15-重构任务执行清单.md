| E0 生命周期 | 悬垂指针 100% 检出 | 无法编译现有 ZHC 代码 | 加阶段性迁移豁免 |
| E1 Result | 错误传播 `?` 语法糖可用 | Result 匹配表达式不完整 | 推迟 Result 到 v1.1 |
| E2 模块系统 | 循环依赖检测通过 | 编译时间增加 > 50% | 优化模块缓存 |

> **安全特性默认开启**，可通过编译选项关闭以兼容纯 C 代码：
> - `-fno-safety-checks`：关闭所有安全检查（与 C 完全兼容）
> - `-fno-null-safety`：仅关闭空指针检查
> - `-fno-bounds-check`：仅关闭边界检查
