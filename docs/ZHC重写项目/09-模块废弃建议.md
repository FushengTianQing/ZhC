# 第九章：模块废弃建议

**版本**: v2.0 | **日期**: 2026-04-12

---

## 9.1 完全废弃的模块

| 模块 | 原位置 | 废弃原因 | 替代方案 |
|:---|:---|:---|:---|
| **C Backend** | `backend/c_backend.py` | 不再生成 C 代码 | LLVM IR 生成 |
| **GCC Backend** | `backend/gcc_backend.py` | 不再调用外部 gcc | LLVM 后端 |
| **Clang Backend** | `backend/clang_backend.py` | 不再调用外部 clang | LLVM 后端 |
| **WASM Backend** | `backend/wasm_backend.py` | 使用 LLVM WASM 目标 | `llc -mtriple=wasm32` |
| **llvmlite Backend** | `backend/llvm_backend.py` | 用 LLVM C++ API 替代 | LLVM C++ API |
| **自定义 IR** | `ir/` | 使用 LLVM IR | LLVM IR |
| **IR 优化** | 无 | 使用 LLVM Pass | PassManager |
| **寄存器分配** | `allocator_interface.py` | 使用 LLVM 寄存器分配器 | LLVM RA |
| **Python Runtime** | `runtime/` | C++ 重写 | `zhc_runtime.cpp` |
| **原始 AST Nodes (Python)** | `parser/ast_nodes.py` | C++ 重写 | `zhc_ast.h` |
| **原始 Lexer (Python)** | `parser/lexer.py` (~1,439行) | C++ 重写 | `zhc_lexer.cpp` |
| **原始 Parser (Python)** | `parser/parser.py` (~2,681行) | C++ 重写 | `zhc_parser.cpp` |

## 9.3 Python 版本维护策略

在 C++ 重写期间，保持 Python 版本维护：

| 阶段 | Python 版本策略 |
|:---|:---|
| **重写期间（P0-P2）** | Python 版本仅做 bug 修复，不新增功能 |
| **C++ 版本 MVP 完成** | Python 版本进入维护模式（仅安全更新） |
| **C++ 版本 v1.0** | Python 版本正式归档，推荐迁移到 C++ 版本 |
| **迁移工具** | 提供 `zhc-migrate` 工具，自动转换 Python 版本代码为 C++ 项目结构 |

## 9.5 关键迁移路径

### 9.5.1 Python 模块 → C++ 模块映射

| Python 模块 | C++ 模块 | 迁移复杂度 |
|:---|:---|:---:|
| `lexer.py` | `zhc_lexer.cpp/h` | 高（~1,439行，含 Token 定义） |
| `parser.py` | `zhc_parser.cpp/h` | 高 |
| `ast_nodes.py` | `zhc_ast.h` | 高（~3,011行，76+ 节点类型） |
| `semantic_analyzer.py` | `zhc_sema.cpp/h` | 高 |
| `ir_generator.py` | `zhc_codegen.cpp/h` | 高 |
| `cli/main.py` | `tools/zhc/zhc.cpp` | 低 |
| `keywords.py` | `zhc_keywords.h` | 低 |

### 9.5.2 数据结构转换

```cpp
// Python dict → C++ std::unordered_map
// Python list → C++ std::vector
// Python tuple → C++ std::array 或 std::pair
// Python set → C++ std::unordered_set

// 举例：Python AST 节点
class ASTNode {
    std::string type;
    std::vector<ASTNode*> children;
    std::unordered_map<std::string, std::string> attributes;
};

// C++ 等价
class ASTNode {
public:
    enum NodeKind { ... };
    NodeKind Kind;
    std::vector<std::unique_ptr<ASTNode>> Children;
    llvm::StringMap<llvm::StringRef> Attributes;
};
```

## 9.7 归档后的维护

### 9.7.1 Python 版本归档仓库

```bash
# 创建归档分支
git branch archive/v1.x-python origin/main
git push origin archive/v1.x-python

# 添加归档标签
git tag -a python-archive-v1.0 -m "Python version archived"
git push origin python-archive-v1.0

# 标记为只读
# (在 GitHub 设置中配置)
```

### 9.7.2 归档后问题处理

| 情况 | 处理方式 |
|:---|:---|
| **安全漏洞** | 接受补丁但不发新版本 |
| **严重 Bug** | 指引迁移到 C++ 版本 |
| **用户请求** | 建议使用 C++ 版本 |
| **贡献者** | 引导贡献到 C++ 版本 |
