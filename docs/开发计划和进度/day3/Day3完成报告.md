### 🚀 实际转换示例

#### 输入 (中文模块)
```c
模块 数学库 {
    公开:
        函数 加法(整数型 a, 整数型 b) -> 整数型 {
            返回 a + b;
        }
        
        整数型 圆周率 = 3;
}
```

#### 输出 (C代码)
**头文件 (`数学库.h`):**
```c
#ifndef __数学库_H__
#define __数学库_H__

#ifdef __cplusplus
extern "C" {
#endif

int 数学库_加法(int a, int b);
int 数学库_圆周率 = 3;

#ifdef __cplusplus
}
#endif

#endif /* __数学库_H__ */
```

**源文件 (`数学库.c`):**
```c
/* 数学库模块 - 自动生成的源文件 */
#include "数学库.h"

int 数学库_加法(int a, int b);
int 数学库_圆周率 = 3;
```
