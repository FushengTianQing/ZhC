# 模块解析器设计文档

## 概述

模块解析器是中文C编译器第三阶段的核心组件，负责解析和转换模块语法。本文档详细说明其设计架构、算法和实现细节。

## 设计目标

### 功能需求
1. **语法解析**: 识别模块声明、导入语句和访问控制关键字
2. **作用域管理**: 管理四级作用域体系
3. **符号转换**: 将中文符号转换为C限定名
4. **依赖分析**: 解析模块间依赖关系
5. **错误处理**: 提供友好的错误提示和恢复机制

### 非功能需求
1. **性能**: 快速解析大型代码库
2. **可扩展性**: 支持未来语法扩展
3. **可维护性**: 清晰的代码结构和文档
4. **兼容性**: 与现有预处理器v3.0保持兼容

## 解析算法

### 1. 词法分析阶段

#### 关键词匹配算法
```python
def match_keyword(text: str) -> Optional[Tuple[str, str]]:
    """
    最大长度优先匹配算法
    返回：(匹配的关键词, 对应的C关键字)
    """
    for keyword in SORTED_KEYS:  # 已按长度降序排序
        if text.startswith(keyword):
            # 检查是否是完整单词
            next_char = text[len(keyword)] if len(text) > len(keyword) else ''
            if not next_char.isalnum() and next_char != '_':
                return keyword, KEYWORDS[keyword]
    return None
```

def parse_module_declaration(line: str) -> Optional[str]:
    match = re.match(MODULE_PATTERN, line.strip())
    return match.group(1) if match else None

#### 导入声明解析
```python
IMPORT_PATTERN = r'导入\s+([\w\s,]+)'

def parse_imports(line: str) -> List[str]:
    match = re.match(IMPORT_PATTERN, line.strip())
    if match:
        return [imp.strip() for imp in match.group(1).split(',')]
    return []
```

#### 符号声明识别
```python
FUNCTION_PATTERN = r'函数\s+(\w+)\s*\(.*\)\s*->\s*\w+\s*\{'
VARIABLE_PATTERN = r'(?:整数型|字符型|浮点型|双精度浮点型)\s+(\w+)\s*(?:=\s*[^;]+)?;'

def extract_symbols(line: str) -> List[str]:
    symbols = []
    
    # 匹配函数声明
    func_match = re.search(FUNCTION_PATTERN, line)
    if func_match:
        symbols.append(func_match.group(1))
        
    # 匹配变量声明
    var_match = re.search(VARIABLE_PATTERN, line)
    if var_match:
        symbols.append(var_match.group(1))
        
    return symbols
```

### 3. 语义分析阶段

#### 作用域管理算法
```python
class ScopeManager:
    def enter_scope(self, name: str, scope_type: ScopeType):
        """进入新作用域"""
        new_scope = Scope(name, scope_type, self.current_scope)
        self.scope_stack.append(new_scope)
        self.current_scope = new_scope
        
    def exit_scope(self):
        """退出当前作用域"""
        if len(self.scope_stack) > 1:
            self.scope_stack.pop()
            self.current_scope = self.scope_stack[-1]
            
    def lookup_symbol(self, name: str) -> Optional[SymbolInfo]:
        """作用域链查找"""
        current = self.current_scope
        while current:
            if name in current.symbols:
                return current.symbols[name]
            current = current.parent
        return None
```

#### 可见性规则
```python
def can_access(symbol: SymbolInfo, from_scope: Scope) -> bool:
    """判断符号是否可访问"""
    
    if symbol.visibility == Visibility.PUBLIC:
        return True  # 公开符号总是可访问
        
    elif symbol.visibility == Visibility.PRIVATE:
        # 私有符号：仅同模块内可访问
        from_module = from_scope.get_module()
        symbol_module = symbol.scope.get_module()
        return from_module == symbol_module
        
    elif symbol.visibility == Visibility.PROTECTED:
        # 保护符号：同模块和子模块可访问
        # TODO: 实现子模块逻辑
        return can_access(symbol, from_scope)  # 暂时与私有相同
        
    return False
```

### 4. 代码生成阶段

#### 符号转换规则
```python
def convert_symbol(symbol: SymbolInfo) -> str:
    """将中文符号转换为C限定名"""
    
    if symbol.scope_type == ScopeType.MODULE:
        # 模块级符号
        module_name = symbol.scope.get_module().name
        
        if symbol.visibility == Visibility.PUBLIC:
            # 公开符号: 模块名_符号名
            return f"{module_name}_{symbol.name}"
        elif symbol.visibility == Visibility.PRIVATE:
            # 私有符号: static 模块名_符号名
            return f"static {module_name}_{symbol.name}"
        elif symbol.visibility == Visibility.PROTECTED:
            # 保护符号: static 模块名_符号名
            return f"static {module_name}_{symbol.name}"
            
    else:
        # 非模块级符号保持不变
        return symbol.name
```

#### 导入处理算法
```python
def process_imports(module: ModuleInfo, all_modules: Dict[str, ModuleInfo]) -> Dict[str, str]:
    """处理模块导入，生成符号映射表"""
    
    symbol_map = {}
    
    for imported_name in module.imports:
        if imported_name not in all_modules:
            raise ImportError(f"模块 '{imported_name}' 未找到")
            
        imported_module = all_modules[imported_name]
        
        # 只导入公开符号
        for symbol in imported_module.public_symbols:
            qualified_name = imported_module.get_qualified_name(symbol)
            symbol_map[symbol] = qualified_name
            
    return symbol_map
```

## 错误处理策略

### 错误分类
1. **语法错误**: 无法识别的关键字或语法结构
2. **语义错误**: 符号重定义、未定义符号引用
3. **导入错误**: 模块不存在或循环依赖
4. **系统错误**: 文件IO错误、内存不足

### 错误恢复机制
```python
class ErrorRecovery:
    def handle_syntax_error(self, line: str, line_num: int):
        """语法错误处理"""
        print(f"语法错误(行{line_num}): {line}")
        # 跳过当前行，继续解析
        
    def handle_semantic_error(self, error_type: str, details: str):
        """语义错误处理"""
        print(f"语义错误[{error_type}]: {details}")
        # 记录错误但继续解析
        
    def panic_mode_recovery(self):
        """恐慌模式恢复"""
        # 跳过大量代码，直到找到可恢复点
        # 如遇到'}'或新的模块声明
```

### 错误消息格式

## 性能优化策略

### 1. 词法分析优化
- **关键词哈希表**: 使用字典实现O(1)查找
- **前缀树(Trie)**: 用于关键词前缀匹配
- **缓存机制**: 缓存已解析的模块信息

### 2. 语法分析优化
- **增量解析**: 只解析修改过的文件
- **并行解析**: 多文件并行处理
- **懒加载**: 按需解析导入的模块

### 3. 内存管理优化
- **对象池**: 重用频繁创建的对象
- **字符串驻留**: 减少重复字符串内存占用
- **流式处理**: 避免一次性加载大文件
