## 二、语法设计

### 2.1 基本类定义
```c
类 学生 {
    属性:
        字符串型 姓名;
        整数型 年龄;
        浮点型 成绩;

    方法:
        函数 构造函数(字符串型 名, 整数型 龄) -> 空型 {
            姓名 = 名;
            年龄 = 龄;
            成绩 = 0.0;
        }

        函数 设置成绩(参数 浮点型 分) -> 空型 {
            成绩 = 分;
        }

        函数 获取信息() -> 字符串型 {
            返回 "学生信息";
        }
}
```

### 2.2 继承语法
```c
类 大学生 : 学生 {
    属性:
        字符串型 专业;
        字符串型 学号;

    方法:
        函数 构造函数(字符串型 名, 整数型 龄, 字符串型 专, 字符串型 号) -> 空型 {
            姓名 = 名;
            年龄 = 龄;
            专业 = 专;
            学号 = 号;
        }
}
```

## 三、转换规则

### 3.1 类转换为结构体
```c
// 中文类
类 学生 { ... }

// 转换后C代码
typedef struct Student {
    char* name;
    int age;
    float score;
} Student;
```

### 3.2 方法转换为函数指针
```c
// 中文方法
函数 设置成绩(参数 浮点型 分) -> 空型 { ... }

// 转换后
void Student_setScore(Student* this, float score);
```

### 3.3 继承转换为嵌套结构体
```c
// 中文继承
类 大学生 : 学生 { ... }

// 转换后C代码
typedef struct Undergraduate {
    Student base;      // 基类
    char* major;       // 派生属性
    char* studentId;
} Undergraduate;
```

## 五、技术要点

### 5.1 核心数据结构
```python
@dataclass
class ClassInfo:
    name: str
    base_class: Optional[str]
    attributes: List[AttributeInfo]
    methods: List[MethodInfo]
    visibility: Visibility
```

### 5.2 转换流程
```
中文类定义 → 解析器识别 → ClassInfo构建 → 结构体生成 → 方法函数生成
```

### 5.3 关键算法
1. **继承链解析**: 递归处理多级继承
2. **虚函数表**: 动态分派实现
3. **this指针**: 自动作为第一个参数添加
