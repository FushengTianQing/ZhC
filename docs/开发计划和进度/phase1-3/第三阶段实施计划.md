## 二、技术方案

### 2.1 核心架构：预处理器扩展方案

延续前两阶段的技术路线，所有新语法在预处理器层（`zhc.py`）转换为标准C代码：

```python
# 架构概览
中文C代码 (.zhc) → 预处理器 (zhc.py) → C代码 (.c) → Clang编译 → 可执行文件
```

### 2.2 模块系统设计

#### 语法示例
```c
模块 数学工具集 {
    公开:
        常量 PI = 3.1415926;
        常量 自然对数的底数 = 2.7182818;
        
        函数 平方(参数 x) -> 整数型 {
            返回 x * x;
        }
        
        函数 立方(参数 x) -> 整数型 {
            返回 x * x * x;
        }
    
    私有:
        静态 整数型 内部计数器 = 0;
}
```

#### 转换规则
```
模块 模块名 { ... } → namespace 模块名 { ... } (C++兼容)
公开: → public: (C++命名空间可见性)
私有: → static (文件内可见)
```

### 2.3 类系统设计

#### 基本语法
```c
类 学生 {
    属性:
        字符串型 姓名;
        整数型 年龄;
        浮点型 成绩;
    
    方法:
        函数 构造函数(字符串型 名, 整数型 龄) -> 空型 {
            姓名 = 名;
            年龄 = 龄;
            成绩 = 0.0;
        }
        
        函数 设置成绩(参数 浮点型 分) -> 空型 {
            成绩 = 分;
        }
        
        函数 获取信息() -> 字符串型 {
            // 注意：实际实现需要字符串格式化函数
            返回 "学生信息";
        }
}

类 大学生 : 学生 {
    属性:
        字符串型 专业;
        字符串型 学号;
}
```

#### 转换规则
```
类 → struct + 函数指针表
属性 → struct成员
方法 → 静态函数（第一个参数为this指针）
继承 → struct嵌套 + 函数转发
```

### 2.4 内存语法设计

#### 内存管理语法
```c
函数 创建数组示例() -> 空型 {
    // 基础内存分配
    整数型* 数组 = 新 整数型[10];
    
    // 智能指针（自动释放）
    智能指针<学生> 小明 = 新 学生("小明", 20);
    小明->设置成绩(85.5);
    
    // 手动释放
    删除 数组;
    // 智能指针自动释放，无需手动删除
}
```

#### 转换规则
```
新 类型 → (类型*)malloc(sizeof(类型))
新 类型[N] → (类型*)malloc(N * sizeof(类型))
删除 ptr → free(ptr)
智能指针<T> → struct + RAII模式
```
